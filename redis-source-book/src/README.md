# Redis 2.0.0 源码解读

## 前言

Redis 是当今最流行的内存数据库之一，由 Salvatore Sanfilippo（antirez）开发。它以高性能、丰富的数据类型和简单的使用方式著称。Redis 的源代码以其简洁优雅的设计闻名，是学习数据库系统和 C 语言编程的绝佳教材。

### 为什么选择 Redis 2.0.0？

Redis 经过了多年的发展，现在的版本已经非常庞大和复杂。而 Redis 2.0.0 是一个相对早期的稳定版本，它包含了 Redis 的核心特性：

- **完整的数据类型**：字符串、列表、集合、有序集合、哈希
- **持久化机制**：RDB 和 AOF
- **主从复制**
- **发布订阅**
- **事务支持**

同时，它的代码量相对较小（约 1 万行），结构清晰，非常适合学习和理解 Redis 的核心设计思想。通过学习 2.0.0 版本，你可以：

1. 理解 Redis 的核心架构和设计哲学
2. 掌握高性能网络编程的技巧
3. 学习常用数据结构的高效实现
4. 了解数据库系统的基本原理

### 本书读者

本书适合以下读者：

- 有一定 C 语言基础的程序员
- 对数据库系统感兴趣的开发者
- 想要深入了解 Redis 内部原理的用户
- 正在准备技术面试的工程师

### 本书结构

本书分为七个部分：

- **第一部分**：介绍 Redis 概述、环境搭建和整体架构
- **第二部分**：深入讲解基础数据结构（SDS、链表、字典、对象系统）
- **第三部分**：分析核心机制（事件驱动、网络层、内存管理）
- **第四部分**：详细解读五种数据类型的命令实现
- **第五部分**：探索高级特性（持久化、复制、发布订阅、事务）
- **第六部分**：理解客户端与协议实现
- **第七部分**：总结与实践

### 如何阅读本书

建议按照章节顺序阅读，因为后面的内容往往依赖于前面介绍的概念。在阅读过程中，强烈建议：

1. 对照源代码阅读
2. 尝试修改代码并观察行为变化
3. 使用调试器跟踪关键函数的执行流程
