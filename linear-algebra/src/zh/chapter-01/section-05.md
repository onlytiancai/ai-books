### 5. 长度（范数）与距离

到目前为止，我们将向量视为具有方向和分量的箭头。为了更有意义地比较它们，我们需要一种方法来描述它们有多“长”，以及它们彼此之间有多“远”。这些概念通过向量的**范数**（norm，即长度）和向量之间的**距离**形式化地定义出来。这些概念将分量的代数运算与空间的几何结构紧密联系在一起。

#### 向量的长度（范数）

向量的范数衡量了其 magnitude（大小），即箭头的长度。
对于 $n$ 维空间中的向量 $v = (v_1, v_2, \ldots, v_n)$，其范数定义为：

$$
\|v\| = \sqrt{v_1^2 + v_2^2 + \cdots + v_n^2}.
$$

这个公式直接来源于**勾股定理**：斜边的长度等于各直角边平方和的平方根。
在二维空间中，这就是原点到某点之间熟悉的距离公式。

**注意**：向量的范数通常用双竖线 $\|v\|$ 表示，以区别于标量的绝对值 $|v|$。

示例：

- 对于 $v = (3, 4)$：  
  $$
  \|v\| = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = 5.
  $$  

- 对于 $w = (1, -2, 2)$：  
  $$
  \|w\| = \sqrt{1^2 + (-2)^2 + 2^2} = \sqrt{1 + 4 + 4} = \sqrt{9} = 3.
  $$

#### 单位向量

**单位向量**是指长度恰好为 1 的向量。
它们非常重要，因为它们只捕捉方向而不包含缩放信息。
要从任何非零向量创建单位向量，只需将其除以其范数：

$$
u = \frac{v}{\|v\|}.
$$

这个过程称为**归一化**（normalization）。

示例：  
对于 $v = (3, 4)$，其单位向量为

$$
u = \left(\tfrac{3}{5}, \tfrac{4}{5}\right).
$$

这与 $(3, 4)$ 指向相同的方向，但长度为 1。

单位向量就像“纯粹的方向”。
它们在投影、定义坐标系和数据标准化中特别有用。

#### 向量之间的距离

两个向量 $u$ 和 $v$ 之间的距离定义为它们之差的长度：

$$
\text{dist}(u, v) = \|u - v\|.
$$

示例：  
设 $u = (2, 1)$ 和 $v = (5, 5)$。那么

$$
u - v = (-3, -4).
$$

其范数为

$$
\sqrt{(-3)^2 + (-4)^2} = \sqrt{9 + 16} = 5.
$$

所以距离是 5。这符合我们的直觉：点 $(2, 1)$ 和 $(5, 5)$ 之间的直线距离。

#### 几何解释

- 范数告诉你一个点距离**原点**有多远。
- 距离告诉你两个点彼此之间有多远。

两者都是用相同的公式计算的——平方和的平方根——只是应用的上下文略有不同。

#### 不同种类的范数

上述公式定义的是**欧几里得范数**（或 $\ell_2$ 范数），这是最常见的一种。
但在线性代数中，其他范数也很有用：

- $\ell_1$ 范数（曼哈顿范数）：  
  $$
  \|v\|_1 = |v_1| + |v_2| + \cdots + |v_n|
  $$  
  （绝对值之和）。

- $\ell_\infty$ 范数（最大范数）：  
  $$
  \|v\|_\infty = \max(|v_1|, |v_2|, \ldots, |v_n|)
  $$  
  （最大分量）。

这些范数改变了“长度”和“距离”的几何形状。例如，在 $\ell_1$ 范数下，单位圆看起来像菱形；在 $\ell_\infty$ 范数下，它看起来像正方形。

#### Python 代码示例：计算不同的范数

我们可以使用 `numpy` 库来轻松计算不同类型的范数。以下代码演示了如何计算向量的 $\ell_2$、$\ell_1$ 和 $\ell_\infty$ 范数。

```python
import numpy as np

# 定义一个向量
v = np.array([3, 4])

# 计算 L2 范数 (欧几里得范数，默认)
norm_l2 = np.linalg.norm(v)
print(f"L2 范数：{norm_l2}")  # 输出：5.0

# 计算 L1 范数 (ord=1)
norm_l1 = np.linalg.norm(v, ord=1)
print(f"L1 范数：{norm_l1}")  # 输出：7.0 (|3| + |4|)

# 计算 L-inf 范数 (ord=np.inf)
norm_linf = np.linalg.norm(v, ord=np.inf)
print(f"L-inf 范数：{norm_linf}")  # 输出：4.0 (max(|3|, |4|))
```

#### 日常类比

- 城市中的步行距离：如果街道呈网格状，$\ell_1$ 范数（绝对差之和）比欧几里得范数更自然。
- 爬楼梯：欧几里得范数就像测量对角线距离，但一步一步走实际上可能是在走 $\ell_1$ 距离。
- 测量数据误差：不同的范数捕捉了预测值与真实值之间“接近程度”的不同概念。欧几里得距离 heavily 惩罚大误差；$\ell_1$ 同等对待所有误差。

#### 代数性质

范数和距离满足关键性质，使它们成为一致的度量工具：

- 非负性：$\|v\| \geq 0$，且仅当 $v = 0$ 时 $\|v\| = 0$。  
- 齐次性：$\|c \cdot v\| = |c| \, \|v\|$（缩放可预测地影响长度）。  
- 三角不等式：$\|u + v\| \leq \|u\| + \|v\|$（直接路径最短）。  
- 对称性（对于距离）：$\text{dist}(u, v) = \text{dist}(v, u)$。  

**提示**：三角不等式直观上意味着“两点之间直线最短”，或者绕路走的距离一定大于等于直接走的距离。

这些性质是范数和距离成为数学中 robust 工具的原因。

#### 为什么这很重要

理解长度和距离是迈向高维几何的第一步。这些概念：

- 允许我们定量地比较向量。
- 构成了角度、正交性和投影等概念的基础。
- 支撑优化问题（例如，“找到最近的向量”是机器学习的核心）。
- 定义了空间的几何结构，根据你使用哪种范数，空间结构会发生显著变化。

#### 练习题解答与解析

以下是上述练习题的详细解答过程及 Python 验证代码。

**1. 计算 (6, 8) 的范数及单位向量**

**解答过程：**
首先计算向量 $v = (6, 8)$ 的 $\ell_2$ 范数：
$$
\|v\| = \sqrt{6^2 + 8^2} = \sqrt{36 + 64} = \sqrt{100} = 10.
$$
接下来，通过将向量除以范数来计算单位向量 $u$：
$$
u = \frac{v}{\|v\|} = \left(\frac{6}{10}, \frac{8}{10}\right) = (0.6, 0.8).
$$
验证单位向量长度：$\sqrt{0.6^2 + 0.8^2} = \sqrt{0.36 + 0.64} = 1$。

**Python 验证：**
```python
v = np.array([6, 8])
norm_v = np.linalg.norm(v)
u = v / norm_v
print(f"范数：{norm_v}, 单位向量：{u}")
```

**2. 求 (1, 1, 1) 和 (4, 5, 6) 之间的距离**

**解答过程：**
设 $a = (1, 1, 1)$，$b = (4, 5, 6)$。
首先计算差向量 $b - a$：
$$
b - a = (4-1, 5-1, 6-1) = (3, 4, 5).
$$
然后计算该差向量的范数：
$$
\text{dist}(a, b) = \|(3, 4, 5)\| = \sqrt{3^2 + 4^2 + 5^2} = \sqrt{9 + 16 + 25} = \sqrt{50} = 5\sqrt{2} \approx 7.07.
$$

**Python 验证：**
```python
a = np.array([1, 1, 1])
b = np.array([4, 5, 6])
dist = np.linalg.norm(b - a)
print(f"距离：{dist}")
```

---

**3. 比较 (0, 0) 和 (3, 4) 的欧几里得距离与曼哈顿距离**

**解答过程：**
设起点为原点 $O=(0,0)$，终点为 $P=(3,4)$。

- **欧几里得距离 ($\ell_2$)**：
  $$
  \|P\|_2 = \sqrt{3^2 + 4^2} = 5.
  $$
  这是两点间的直线距离。

- **曼哈顿距离 ($\ell_1$)**：
  $$
  \|P\|_1 = |3| + |4| = 7.
  $$
  这是在网格状街道上必须行走的距离（先走 3 个街区，再走 4 个街区）。

**结论**：如果你沿着城市网格行走（不能穿过建筑物），**曼哈顿距离**更符合直觉。欧几里得距离假设你可以直接“飞”过去。

**Python 验证：**
```python
p = np.array([3, 4])
dist_l2 = np.linalg.norm(p)
dist_l1 = np.linalg.norm(p, ord=1)
print(f"欧几里得距离：{dist_l2}, 曼哈顿距离：{dist_l1}")
```

**4. 挑战：计算 u = (2, –1, 3) 和 v = (–2, 0, 1) 的 ‖u – v‖ 及其几何意义**

**解答过程：**
首先计算差向量 $u - v$：
$$
u - v = (2 - (-2), -1 - 0, 3 - 1) = (4, -1, 2).
$$
然后计算该向量的范数：
$$
\|u - v\| = \sqrt{4^2 + (-1)^2 + 2^2} = \sqrt{16 + 1 + 4} = \sqrt{21} \approx 4.58.
$$

**几何意义**：
这个数值代表三维空间中，点 $(2, -1, 3)$ 与点 $(-2, 0, 1)$ 之间的**直线距离**。它量化了这两个向量尖端之间的空间间隔。

**Python 验证：**
```python
u = np.array([2, -1, 3])
v = np.array([-2, 0, 1])
dist_uv = np.linalg.norm(u - v)
print(f"向量 u 和 v 之间的距离：{dist_uv}")
```
