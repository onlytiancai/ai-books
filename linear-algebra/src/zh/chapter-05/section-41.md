### 41. 线性变换

线性变换是线性代数的核心概念。它是连接两个向量空间的规则，并且尊重它们的线性结构：即加法和标量乘法。与其将向量视为静态对象，不如通过线性变换来研究向量如何移动、拉伸、旋转、投影或反射。线性变换赋予了线性代数动态的力量，它是连接抽象理论与具体应用的桥梁。

#### 定义

向量空间之间的函数 $T: V \to W$ 被称为线性变换，如果对于所有的 $u, v \in V$ 和标量 $a, b \in \mathbb{R}$（或其他域），满足：

$$
T(au + bv) = aT(u) + bT(v).
$$

这一个条件编码了两条基本规则：

1. 可加性 (Additivity)：$T(u+v) = T(u) + T(v)$。
2. 齐次性 (Homogeneity)：$T(av) = aT(v)$。

如果同时满足这两条，该变换即为线性的。

> **注意**：线性变换必须保持原点不变。即 $T(0) = 0$。如果一个变换移动了原点（例如平移），它就不是线性的。

#### 线性变换示例

1. 缩放 (Scaling)：
   $T(x) = 3x$ 在 $\mathbb{R}$ 中。每个数都被拉伸为原来的三倍。

2. 平面旋转 (Rotation in the plane)：
   $T(x,y) = (x\cos\theta - y\sin\theta, \, x\sin\theta + y\cos\theta)$。

3. 投影 (Projection)：
   将 $(x,y,z)$ 投影到 $xy$-平面：
   $T(x,y,z) = (x,y,0)$。

4. 微分 (Differentiation)：
   在多项式空间上，$T(p(x)) = p'(x)$。

5. 积分 (Integration)：
   在连续函数空间上，$T(f)(x) = \int_0^x f(t) \, dt$。

所有这些变换都是线性的，因为它们保持了加法和缩放性质。

#### 非示例

- $T(x) = x^2$ 不是线性的，因为 $(x+y)^2 \neq x^2 + y^2$。
- $T(x,y) = (x+1, y)$ 不是线性的，因为它不满足齐次性：缩放无法保留"+1"这一项。

非线性规则破坏了向量空间的结构。

#### 矩阵表示

从 $\mathbb{R}^n$ 到 $\mathbb{R}^m$ 的每一个线性变换都可以用一个矩阵来表示。

如果 $T: \mathbb{R}^n \to \mathbb{R}^m$，那么存在一个 $m \times n$ 矩阵 $A$ 使得：

$$
T(x) = Ax.
$$

矩阵 $A$ 的列向量 simply 就是 $T(e_1), T(e_2), \dots, T(e_n)$，其中 $e_i$ 是标准基向量。

**示例**：
设 $T(x,y) = (2x+y, x-y)$。

- $T(e_1) = T(1,0) = (2,1)$。
- $T(e_2) = T(0,1) = (1,-1)$。
  所以

$$
A = \begin{bmatrix} 2 & 1 \\ 1 & -1 \end{bmatrix}.
$$

那么 $T(x,y) = A \begin{bmatrix} x \\ y \end{bmatrix}$。

> **提示**：要找到线性变换的矩阵，只需计算基向量在变换后的结果，并将它们作为矩阵的列排列即可。

**Python 代码示例**：
我们可以使用 `numpy` 来验证上述矩阵表示是否正确计算了线性变换。

```python
import numpy as np

# 定义变换矩阵 A
A = np.array([[2, 1],
              [1, -1]])

# 定义输入向量 v = (3, 4)
v = np.array([3, 4])

# 方法 1: 使用矩阵乘法计算 T(v)
T_v_matrix = A @ v

# 方法 2: 直接使用变换公式计算 T(v) = (2x+y, x-y)
T_v_formula = np.array([2*v[0] + v[1], v[0] - v[1]])

print(f"输入向量 v: {v}")
print(f"矩阵计算结果 T(v): {T_v_matrix}")
print(f"公式计算结果 T(v): {T_v_formula}")
print(f"结果是否一致：{np.allclose(T_v_matrix, T_v_formula)}")

# 验证线性性质 T(u+v) = T(u) + T(v)
u = np.array([1, 2])
w = np.array([2, 1])
lhs = A @ (u + w)
rhs = A @ u + A @ w
print(f"可加性验证 T(u+v) == T(u)+T(v): {np.allclose(lhs, rhs)}")
```

#### 线性变换的性质

1. 零向量的像始终为零：$T(0) = 0$。
2. 过原点的直线的像仍然是一条直线（或坍缩为一个点）。
3. 线性变换的复合仍然是线性的。
4. 每个线性变换都保持子空间的结构。

#### 核与像（预览）

对于 $T: V \to W$：

- 核 (Kernel，或零空间) 是所有映射到零的向量：$\ker T = \{v \in V : T(v) = 0\}$。
- 像 (Image，或值域) 是所有可以达到的输出：$\text{im}(T) = \{T(v) : v \in V\}$。
  秩 - 零化度定理 (Rank–Nullity Theorem) 在此适用：

$$
\dim(\ker T) + \dim(\text{im}(T)) = \dim(V).
$$

#### 几何解释

线性变换重塑空间：

- 缩放 (Scaling) 沿一个方向均匀拉伸空间。
- 旋转 (Rotation) 旋转空间同时保持长度不变。
- 投影 (Projection) 将空间扁平化到更低维度。
- 反射 (Reflection) 沿直线或平面翻转空间。

关键特征：直线保持为直线，且原点保持固定。

#### 日常类比

- 地图与坐标：GPS 系统将地理位置（经纬度）转换为屏幕位置（x,y）。
- 影子：将 3D 物体投影到地面上是一种线性变换。
- 经济学：一个将生产水平（输入）转化为利润（输出）的矩阵是数据的线性变换。
- 音乐：均衡器线性地调整不同频率：声音的结构被保留但被重塑。

#### 应用领域

1. 计算机图形学：缩放、旋转、将 3D 物体投影到 2D 屏幕。
2. 机器人学：关节坐标与工作空间位置之间的变换。
3. 数据科学：线性映射表示降维和特征提取。
4. 微分方程：解通常涉及作用于函数空间的线性算子。
5. 机器学习：神经网络中的权重矩阵是堆叠的线性变换，其间穿插非线性激活函数。

#### 为什么它很重要

1. 线性变换将矩阵的概念推广到任何向量空间。
2. 它们在一个概念下统一了几何、代数和应用。
3. 它们为研究特征值、特征向量和分解提供了自然框架。
4. 它们建模了无数现实世界的过程：物理的、计算的和抽象的。

#### 参考答案与详细解析

**1. 证明 $T(x,y,z) = (x+2y, z, x-y+z)$ 是线性的。**

**解答过程**：
我们需要验证可加性和齐次性。设 $u = (x_1, y_1, z_1)$，$v = (x_2, y_2, z_2)$，标量为 $c$。

*   **可加性**：
    $$
    \begin{aligned}
    T(u+v) &= T(x_1+x_2, y_1+y_2, z_1+z_2) \\
    &= ((x_1+x_2)+2(y_1+y_2), \, z_1+z_2, \, (x_1+x_2)-(y_1+y_2)+(z_1+z_2)) \\
    &= (x_1+2y_1, z_1, x_1-y_1+z_1) + (x_2+2y_2, z_2, x_2-y_2+z_2) \\
    &= T(u) + T(v)
    \end{aligned}
    $$
    可加性成立。

*   **齐次性**：
    $$
    \begin{aligned}
    T(cu) &= T(cx_1, cy_1, cz_1) \\
    &= (cx_1+2cy_1, \, cz_1, \, cx_1-cy_1+cz_1) \\
    &= c(x_1+2y_1, \, z_1, \, x_1-y_1+z_1) \\
    &= cT(u)
    \end{aligned}
    $$
    齐次性成立。

**结论**：因为同时满足可加性和齐次性，所以 $T$ 是线性变换。

**2. 找出将 $\mathbb{R}^2$ 中的向量关于直线 $y=x$ 反射的变换的矩阵表示。**

**解答过程**：
我们需要找到标准基向量 $e_1 = (1,0)$ 和 $e_2 = (0,1)$ 在变换后的位置。
关于直线 $y=x$ 反射意味着交换 $x$ 和 $y$ 坐标。

-   $T(e_1) = T(1,0) = (0,1)$。
-   $T(e_2) = T(0,1) = (1,0)$。

将这两个结果作为矩阵的列：
第一列是 $\begin{bmatrix} 0 \\ 1 \end{bmatrix}$，第二列是 $\begin{bmatrix} 1 \\ 0 \end{bmatrix}$。

**结论**：矩阵为
$$
A = \begin{bmatrix} 0 & 1 \\ 1 & 0 \end{bmatrix}.
$$

**3. 说明为什么 $T(x,y) = (x^2,y)$ 不是线性的。**

**解答过程**：
我们要寻找一个反例来证明它不满足线性性质（通常检查齐次性最简单）。
设 $v = (1, 1)$，标量 $c = 2$。

-   计算 $T(cv)$：
    $$
    T(2v) = T(2, 2) = (2^2, 2) = (4, 2).
    $$
-   计算 $cT(v)$：
    $$
    2T(1, 1) = 2(1^2, 1) = 2(1, 1) = (2, 2).
    $$

显然 $(4, 2) \neq (2, 2)$，即 $T(cv) \neq cT(v)$。

**结论**：由于不满足齐次性，$T$ 不是线性变换。

**4. 挑战：对于微分算子 $D: P_3 \to P_2$，找出其矩阵。**

**解答过程**：
域基底为 $\mathcal{B} = \{1, x, x^2, x^3\}$，陪域基底为 $\mathcal{C} = \{1, x, x^2\}$。
我们需要计算每个域基向量在 $D$ 下的像，并将其表示为陪域基底的线性组合。

-   $D(1) = 0 = 0\cdot 1 + 0\cdot x + 0\cdot x^2 \implies$ 坐标向量 $\begin{bmatrix} 0 \\ 0 \\ 0 \end{bmatrix}$
-   $D(x) = 1 = 1\cdot 1 + 0\cdot x + 0\cdot x^2 \implies$ 坐标向量 $\begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix}$
-   $D(x^2) = 2x = 0\cdot 1 + 2\cdot x + 0\cdot x^2 \implies$ 坐标向量 $\begin{bmatrix} 0 \\ 2 \\ 0 \end{bmatrix}$
-   $D(x^3) = 3x^2 = 0\cdot 1 + 0\cdot x + 3\cdot x^2 \implies$ 坐标向量 $\begin{bmatrix} 0 \\ 0 \\ 3 \end{bmatrix}$

将这些坐标向量作为矩阵的列。注意矩阵大小为 $3 \times 4$（陪域维度 $\times$ 域维度）。

**结论**：矩阵为
$$
A = \begin{bmatrix}
0 & 1 & 0 & 0 \\
0 & 0 & 2 & 0 \\
0 & 0 & 0 & 3
\end{bmatrix}.
$$

**Python 验证代码（针对第 4 题）**：
```python
import numpy as np

# 定义微分算子的矩阵表示
D_matrix = np.array([
    [0, 1, 0, 0],
    [0, 0, 2, 0],
    [0, 0, 0, 3]
])

# 定义一个多项式 p(x) = 2 + 3x + 4x^2 + 5x^3
# 其坐标向量为 [2, 3, 4, 5]^T (对应基底 1, x, x^2, x^3)
p_coords = np.array([2, 3, 4, 5])

# 计算导数的坐标向量
dp_coords = D_matrix @ p_coords

print(f"多项式系数 (1, x, x^2, x^3): {p_coords}")
print(f"导数多项式系数 (1, x, x^2): {dp_coords}")
# 预期结果：D(p) = 3 + 8x + 15x^2
# 对应系数应为 [3, 8, 15]
```

线性变换是线性代数的语言。它们捕捉了任何空间中的对称性、运动和结构的本质，使其成为理论和实践不可或缺的工具。
