### 36. 维数 (Dimension)

维数是线性代数中最深刻、最具统一性的概念之一。它用一个单一的数字捕捉了向量空间的“大小”或“容量”：即它拥有多少个独立的方向。与日常几何中的长、宽、高不同，线性代数中的维数适用于任何类型的空间——无论是几何空间、代数空间，甚至是函数空间。

#### 定义

向量空间 $V$ 的**维数**（dimension）是 $V$ 的任意一组基中所含向量的个数。

- 由于向量空间的所有基都含有相同数量的元素，因此维数是良定义的（well-defined）。
- 如果 $\dim V = n$，那么：

  - $V$ 中任何包含多于 $n$ 个向量的集合都是线性相关的。
  - $V$ 中任何包含恰好 $n$ 个线性无关向量的集合都构成一组基。

> **注意**：维数是一个整数。零空间 $\{0\}$ 的维数定义为 0，因为它没有基（或者说基为空集）。

#### 常见空间中的例子

1. $\dim(\mathbb{R}^2) = 2$.

   - 基：$(1,0), (0,1)$.
   - 两个独立方向覆盖了整个平面。

2. $\dim(\mathbb{R}^3) = 3$.

   - 基：$(1,0,0), (0,1,0), (0,0,1)$.
   - 三个独立方向张成了 3D 空间。

3. 所有次数 ≤ 2 的多项式构成的集合，其维数为 3.

   - 基：$\{1, x, x^2\}$.
   - 任意二次多项式 $ax^2 + bx + c$ 都可以由这三个基向量线性表示。

4. 所有 $m \times n$ 矩阵构成的空间，其维数为 $mn$.

   - 每个元素都是独立的，标准基由仅在某个位置为 1、其余位置为 0 的矩阵组成。

#### 有限维与无限维

- **有限维空间**：如 $\mathbb{R}^n$，次数 ≤ $k$ 的多项式空间。
- **无限维空间**：

  - 所有多项式构成的空间（没有次数限制）。
  - 所有连续函数构成的空间。
  - 这些空间无法由有限个向量张成。

> **提示**：在大多数 introductory 线性代数课程中，我们主要关注有限维空间。无限维空间通常在泛函分析中深入研究。

#### 维数与子空间

- $\mathbb{R}^n$ 的任何子空间，其维数 ≤ $n$.
- $\mathbb{R}^3$ 中过原点的直线：维数为 1.
- $\mathbb{R}^3$ 中过原点的平面：维数为 2.
- 整个空间：维数为 3.
- 零子空间 $\{0\}$：维数为 0.

#### 维数与方程组

当求解 $A\mathbf{x} = \mathbf{b}$ 时：

- 列空间的维数 = 秩 (rank) = 输出中独立方向的数量。
- 零空间的维数 = 自由变量的个数。
- 根据**秩 - 零化度定理 (Rank-Nullity Theorem)**：

  $$
  \dim(\text{column space}) + \dim(\text{null space}) = \text{number of variables}.
  $$

  即：$\text{rank}(A) + \text{nullity}(A) = n$（其中 $n$ 是列数）。

#### 几何意义

- 维数统计了描述一个向量所需的最少坐标数量。
- 在 $\mathbb{R}^2$ 中，你需要 2 个数字。
- 在 $\mathbb{R}^3$ 中，你需要 3 个数字。
- 在次数 ≤ 3 的多项式空间中，你需要 4 个系数。

因此，**维数 = 坐标列表的长度**。

#### 日常类比

- **地图**：平面地图需要 2 个坐标（纬度、经度）。地球仪是 3D 的，需要 3 个坐标。加上海拔高度也是 3D。
- **语言**：要描述含义，你需要足够多的独立词汇。冗余的词汇不会增加维数，只有新的独立概念才会。
- **食谱**：要描述所有可能的口味，你需要一定数量的独立食材。更多食材 = 更高维的“口味空间”。

#### 实践中检查维数

1. 将候选向量作为矩阵的列放置。
2. 行化简为 echelon form（行阶梯形）。
3. 数主元（pivots）的个数。该数字 = 这些向量张成空间的维数。

下面是一个使用 Python `numpy` 库来验证向量组维数（即矩阵的秩）的示例：

```python
import numpy as np

# 定义三个向量
v1 = np.array([1, 2, 3])
v2 = np.array([2, 4, 6])  # v2 是 v1 的 2 倍
v3 = np.array([0, 0, 0])  # 零向量

# 将它们作为列构成矩阵
A = np.column_stack((v1, v2, v3))

# 计算矩阵的秩 (Rank)，即列空间的维数
dimension = np.linalg.matrix_rank(A)

print(f"矩阵:\n{A}")
print(f"向量组张成空间的维数 (秩): {dimension}")
# 预期输出：1，因为只有 v1 是独立的
```

#### 为什么重要

1. 维数是向量空间最基本的度量。
2. 它告诉我们空间有多“大”或多“复杂”。
3. 它设定了绝对限制：在 $\mathbb{R}^n$ 中，不存在多于 $n$ 个的线性无关向量。
4. 它是坐标系、基和变换的基础。
5. 它架起了几何（线、面、体）与代数（解、方程、矩阵）之间的桥梁。

### 练习题解答与详细过程

#### 1. 向量 $(1,2,3)$, $(2,4,6)$, $(0,0,0)$ 张成空间的维数是多少？

**解答：**
维数为 **1**。

**过程：**
1.  观察向量之间的关系。
2.  第二个向量 $(2,4,6)$ 是第一个向量 $(1,2,3)$ 的 2 倍，即 $(2,4,6) = 2 \times (1,2,3)$。因此它们是线性相关的。
3.  第三个向量是零向量 $(0,0,0)$。零向量不增加维数，因为它可以由任何向量乘以 0 得到。
4.  因此，这组向量中只有一个线性无关的向量（即 $(1,2,3)$）。
5.  基可以取为 $\{(1,2,3)\}$，基中向量个数为 1，故维数为 1。

#### 2. 求 $\mathbb{R}^3$ 中由 $x+y+z=0$ 定义的子空间的维数。

**解答：**
维数为 **2**。

**过程：**
1.  该子空间是方程 $x+y+z=0$ 的解空间。
2.  我们可以将方程改写为 $x = -y - z$。
3.  这里 $y$ 和 $z$ 可以作为自由变量。
4.  任意解向量 $\mathbf{v}$ 可以表示为：
    $$
    \mathbf{v} = \begin{pmatrix} x \\ y \\ z \end{pmatrix} = \begin{pmatrix} -y - z \\ y \\ z \end{pmatrix} = y \begin{pmatrix} -1 \\ 1 \\ 0 \end{pmatrix} + z \begin{pmatrix} -1 \\ 0 \\ 1 \end{pmatrix}
    $$
5.  这两个向量 $\begin{pmatrix} -1 \\ 1 \\ 0 \end{pmatrix}$ 和 $\begin{pmatrix} -1 \\ 0 \\ 1 \end{pmatrix}$ 是线性无关的。
6.  它们构成了该子空间的一组基，基中含有 2 个向量。
7.  几何上，这是 $\mathbb{R}^3$ 中过原点的一个平面，平面的维数自然是 2。

#### 3. 证明所有 $2 \times 2$ 对称矩阵构成的集合维数为 3。

**解答：**
维数为 **3**。

**过程：**
1.  一个 $2 \times 2$ 对称矩阵 $A$ 满足 $A^T = A$，其形式必须为：
    $$
    A = \begin{pmatrix} a & b \\ b & c \end{pmatrix}
    $$
    其中 $a, b, c$ 是任意实数。
2.  我们可以将该矩阵分解为三个基本矩阵的线性组合：
    $$
    \begin{pmatrix} a & b \\ b & c \end{pmatrix} = a \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix} + b \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix} + c \begin{pmatrix} 0 & 0 \\ 0 & 1 \end{pmatrix}
    $$
3.  令 $E_1 = \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix}$, $E_2 = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$, $E_3 = \begin{pmatrix} 0 & 0 \\ 0 & 1 \end{pmatrix}$。
4.  显然，$E_1, E_2, E_3$ 是线性无关的（因为每个矩阵中非零元素的位置不同，无法互相表示）。
5.  这三个矩阵构成了对称矩阵空间的一组基。
6.  基中向量个数为 3，故维数为 3。

#### 4. 挑战：证明次数 ≤ $k$ 的多项式空间维数为 $k+1$。

**解答：**
维数为 **$k+1$**。

**过程：**
1.  记该空间为 $P_k$。任意多项式 $p(x) \in P_k$ 可以写成：
    $$
    p(x) = a_0 + a_1 x + a_2 x^2 + \dots + a_k x^k
    $$
    其中 $a_0, \dots, a_k$ 是系数。
2.  考虑集合 $B = \{1, x, x^2, \dots, x^k\}$。
3.  **张成性**：任何 $P_k$ 中的多项式都可以由 $B$ 中的元素线性表示（见上述公式）。
4.  **线性无关性**：假设存在系数 $c_0, \dots, c_k$ 使得：
    $$
    c_0 \cdot 1 + c_1 \cdot x + \dots + c_k \cdot x^k = 0 \quad (\text{零多项式})
    $$
    根据多项式恒等定理，若多项式恒等于 0，则其所有系数必须为 0。即 $c_0 = c_1 = \dots = c_k = 0$。因此 $B$ 是线性无关的。
5.  因此，$B$ 是 $P_k$ 的一组基。
6.  基中元素的个数是从 $x^0$ 到 $x^k$，共 $k+1$ 个。
7.  故 $\dim(P_k) = k+1$。

---

维数是线性代数的测量尺：它告诉我们需要多少独立的信息片段来描述整个空间。
