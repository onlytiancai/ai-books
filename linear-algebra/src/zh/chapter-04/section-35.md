### 35. 基与坐标

张成（Span）和线性无关（Linear Independence）这两个核心概念汇聚成了一个强大的思想——**基（Basis）**。基为我们提供了生成整个向量空间所需的最少“积木”，且没有任何冗余。一旦选定了一组基，空间中的每一个向量都可以通过一组唯一的数字来描述，这组数字被称为该向量在这组基下的**坐标**。

#### 什么是基？

向量空间 $V$ 的一组基是向量集合 $\{v_1, v_2, \dots, v_k\}$，它必须满足以下两个性质：

1.  **生成性（Spanning property）**：$\text{span}\{v_1, \dots, v_k\} = V$。
2.  **无关性（Independence property）**：这些向量是线性无关的。

简而言之：**基是一个没有冗余的生成集。**

> **注意**：基不是唯一的。同一个向量空间可以有无数多组不同的基，但所有基包含的向量个数是相同的。

#### 示例：标准基

1.  在 $\mathbb{R}^2$ 中，标准基是 $\{(1,0), (0,1)\}$。
2.  在 $\mathbb{R}^3$ 中，标准基是 $\{(1,0,0), (0,1,0), (0,0,1)\}$。
3.  在 $\mathbb{R}^n$ 中，标准基是一组单位向量的集合，每个向量在一个位置上是 1，其余位置是 0。

之所以称为“标准”，是因为它们是我们描述坐标时的默认方式。

#### 坐标的唯一性

关于基最重要的事实之一是：它们提供了向量的**唯一表示**。

- 给定一组基 $\{v_1, \dots, v_k\}$，空间中的任意向量 $x \in V$ 都可以被**唯一地**写为：

  $$
  x = a_1 v_1 + a_2 v_2 + \dots + a_k v_k.
  $$
- 系数 $(a_1, a_2, \dots, a_k)$ 就是 $x$ 相对于该基的**坐标**。

这种唯一性将基与普通的生成集区分开来。普通的生成集可能存在冗余，导致同一个向量有多种不同的表示方式。

> **提示**：线性无关性保证了表示的唯一性。如果向量线性相关，那么零向量就有非零的线性组合表示，从而导致其他向量的表示也不唯一。

#### $\mathbb{R}^2$ 中的示例

设基 = $\{(1,0), (0,1)\}$。

- 向量 $(3,5) = 3(1,0) + 5(0,1)$。
- 相对于该基的坐标：$(3,5)$。

如果我们换一组基，即使向量本身不变，其坐标也会发生变化。

#### 非标准基示例

设 $\mathbb{R}^2$ 中的基 = $\{(1,1), (1,-1)\}$。
求 $x = (2,0)$ 的坐标。

我们需要求解 $a(1,1) + b(1,-1) = (2,0)$。
这给出了以下方程组：

$$
a + b = 2, \quad a - b = 0.
$$

解得 $a=1, b=1$。
相对于该基的坐标：$(1,1)$。

注意：坐标依赖于基的选择。同一个几何向量，在不同基下的坐标是不同的。

#### Python 数值演示：计算坐标

我们可以使用 `numpy` 来求解线性方程组，从而找到向量在非标准基下的坐标。这相当于求解矩阵方程 $Ax=b$，其中 $A$ 的列是基向量，$b$ 是目标向量。

```python
import numpy as np

# 定义基向量 (作为矩阵的列)
# 基 = {(1,1), (1,-1)}
basis_vectors = np.array([[1, 1], 
                          [1, -1]])

# 定义目标向量 x = (2,0)
target_vector = np.array([2, 0])

# 求解线性方程组 basis_vectors * coordinates = target_vector
# np.linalg.solve 求解 Ax = b
coordinates = np.linalg.solve(basis_vectors, target_vector)

print(f"基向量矩阵:\n{basis_vectors}")
print(f"目标向量: {target_vector}")
print(f"计算出的坐标: {coordinates}")
# 输出应接近 [1. 1.]
```

#### 函数空间的基

基的概念不仅限于几何向量，也适用于函数空间。

1.  对于次数 ≤ 2 的多项式空间：基 = $\{1, x, x^2\}$。

   - 示例：多项式 $2 + 3x + 5x^2$ 的坐标为 $(2,3,5)$。
2.  对于 $[0,1]$ 上的连续函数，一个可能的基是无限集合 $\{1, x, x^2, \dots\}$。

这表明基的概念并不局限于几何向量，它可以推广到更抽象的空间。

#### 维数

基中向量的个数称为向量空间的**维数（Dimension）**。

- $\mathbb{R}^2$ 的维数是 2。
- $\mathbb{R}^3$ 的维数是 3。
- 次数 ≤ 3 的多项式空间的维数是 4（基为 $\{1, x, x^2, x^3\}$）。

维数告诉我们空间中存在多少个独立的方向。

> **注意**：虽然基不唯一，但向量空间的维数是唯一的。这是线性代数中的一个基本定理。

#### 基变换

- 从一组基切换到另一组基，就像在两种语言之间进行翻译。
- 同一个向量取决于你使用哪本“字典”（基），看起来会有所不同。
- **基变换矩阵**允许我们在不同的坐标系统之间进行系统的转换。

#### 几何解释

- 基就像是在空间中建立坐标轴。
- 在 2D 中，两个无关向量定义了一个网格。
- 在 3D 中，三个无关向量定义了一个完整的坐标系。
- 不同的基 = 覆盖在同一空间上的不同网格。

#### 日常类比

- **语言**：基就像字母表。每个单词（向量）都可以由字母（基向量）唯一拼写出来。
- **颜色**：RGB 是颜色空间的一组基。任何颜色都可以通过混合红、绿、蓝唯一描述。
- **音乐**：音符构成了一组基。每个和弦或旋律都是它们的组合。

#### 为什么它很重要

1.  基提供了描述向量空间的最简单方式。
2.  它们允许我们为向量分配唯一的坐标。
3.  它们将空间的抽象结构与具体的数值表示联系起来。
4.  这一概念是几乎所有线性代数内容的基础：维数、变换、特征向量等等。

#### 练习题详细解答

**1. 证明 $\{(1,2), (3,4)\}$ 是 $\mathbb{R}^2$ 的一组基。**

**解答过程：**
要证明这是一组基，需要验证两个条件：生成性和线性无关性。对于 $\mathbb{R}^n$ 中的 $n$ 个向量，如果它们线性无关，则自动生成该空间。
我们将这两个向量组成矩阵 $A$，计算其行列式。如果行列式不为零，则向量线性无关。

$$
A = \begin{pmatrix} 1 & 3 \\ 2 & 4 \end{pmatrix}
$$

计算行列式：
$$
\det(A) = (1)(4) - (3)(2) = 4 - 6 = -2 \neq 0.
$$

**结论：** 因为行列式非零，向量线性无关。又因为向量个数（2 个）等于空间维数（2 维），所以它们构成 $\mathbb{R}^2$ 的一组基。

**Python 验证：**
```python
import numpy as np

v1 = np.array([1, 2])
v2 = np.array([3, 4])
matrix = np.column_stack((v1, v2))

# 计算行列式
det = np.linalg.det(matrix)
print(f"行列式: {det}") # 输出应接近 -2.0

# 检查秩是否等于维数
rank = np.linalg.matrix_rank(matrix)
print(f"矩阵的秩: {rank}") # 输出应为 2
```

**2. 用基 $\{(1,1), (1,-1)\}$ 表示向量 $(4,5)$。**

**解答过程：**
设坐标为 $(a, b)$，我们需要求解：
$$
a(1,1) + b(1,-1) = (4,5)
$$
展开得到线性方程组：
$$
\begin{cases}
a + b = 4 \\
a - b = 5
\end{cases}
$$
将两式相加：
$$
2a = 9 \implies a = 4.5
$$
将 $a$ 代入第一式：
$$
4.5 + b = 4 \implies b = -0.5
$$

**结论：** 相对于该基，向量 $(4,5)$ 的坐标为 $(4.5, -0.5)$。

**3. 证明 $\mathbb{R}^3$ 的任何基都不能包含超过 3 个向量。**

**解答过程：**
这是基于维数定理的一个基本性质。
1.  **定义**：$\mathbb{R}^3$ 的维数是 3，意味着其标准基包含 3 个向量。
2.  **定理**：在 $n$ 维向量空间中，任何包含超过 $n$ 个向量的集合必然是线性相关的。
3.  **推导**：假设 $\mathbb{R}^3$ 有一组基包含 $k$ 个向量，且 $k > 3$。根据基的定义，这组向量必须线性无关。然而，根据上述定理，$\mathbb{R}^3$ 中任意 4 个或更多向量必然线性相关。
4.  **矛盾**：这就产生了矛盾（既要求无关，又必然相关）。

**结论：** 因此，$\mathbb{R}^3$ 的基最多只能包含 3 个向量。实际上，任何基必须恰好包含 3 个向量。

**4. 挑战：证明集合 $\{1, \cos x, \sin x\}$ 是由 $1, \cos x, \sin x$ 的所有线性组合构成的空间的一组基。**

**解答过程：**
设该空间为 $V = \text{span}\{1, \cos x, \sin x\}$。
根据定义，这组集合显然生成 $V$。我们需要证明它们是**线性无关**的。

假设存在常数 $c_1, c_2, c_3$ 使得对于所有 $x$ 都有：
$$
c_1 \cdot 1 + c_2 \cos x + c_3 \sin x = 0
$$
我们需要证明 $c_1 = c_2 = c_3 = 0$。

我们可以通过选取特殊的 $x$ 值来构建方程组：
1.  取 $x = 0$：
    $$
    c_1 + c_2(1) + c_3(0) = 0 \implies c_1 + c_2 = 0 \quad \text{(式 1)}
    $$
2.  取 $x = \pi$：
    $$
    c_1 + c_2(-1) + c_3(0) = 0 \implies c_1 - c_2 = 0 \quad \text{(式 2)}
    $$
    由 (式 1) 和 (式 2) 可得：$2c_1 = 0 \implies c_1 = 0$，进而 $c_2 = 0$。
3.  取 $x = \frac{\pi}{2}$：
    $$
    c_1(1) + c_2(0) + c_3(1) = 0 \implies c_1 + c_3 = 0
    $$
    因为已知 $c_1 = 0$，所以 $c_3 = 0$。

**结论：** 只有当所有系数为零时，线性组合才恒为零。因此，函数集 $\{1, \cos x, \sin x\}$ 线性无关，构成该函数空间的一组基。

---

基是向量空间最小且优雅的基础，通过提供一组有限的独立构建块，它将无限的可能性变得可管理。
