### 31. 向量空间的公理

在此之前，我们主要在 $\mathbb{R}^2$、$\mathbb{R}^3$ 以及更高维的欧几里得空间中处理向量。但线性代数的真正力量在于**脱离坐标的抽象**。向量空间并不局限于物理空间中的箭头——它是任何表现为向量行为的对象集合，只要它们满足特定的规则。这些规则被称为**向量空间公理**。

#### 向量空间的概念

一个向量空间是一个集合 $V$，配备了两种运算：

1.  **向量加法**：将 $V$ 中的两个向量组合，得到 $V$ 中的另一个向量。
2.  **标量乘法**：将 $V$ 中的一个向量乘以一个标量（来自某个域的数，通常是 $\mathbb{R}$ 或 $\mathbb{C}$）。

神奇之处在于，只要某些规则（公理）成立，$V$ 中的对象就可以被视为向量。它们不必是箭头或坐标列表——它们可以是多项式、函数、矩阵或序列。

> **注意**：在中国国内的教材中（如同济大学《线性代数》），通常将“封闭性”隐含在运算的定义中，因此常列举为 **8 条公理**。本文为了概念清晰，将封闭性单独列出，共 10 条，但本质是一致的。

#### 十条公理

设 $u, v, w \in V$（向量）且 $a, b \in \mathbb{R}$（标量）。公理如下：

1.  **加法封闭性**：$u + v \in V$。
2.  **加法交换律**：$u + v = v + u$。
3.  **加法结合律**：$(u + v) + w = u + (v + w)$。
4.  **加法单位元存在性**：存在一个零向量 $0 \in V$，使得 $v + 0 = v$。
5.  **加法逆元存在性**：对于每个 $v$，存在 $-v$ 使得 $v + (-v) = 0$。
6.  **标量乘法封闭性**：$a v \in V$。
7.  **标量乘法对向量加法的分配律**：$a(u + v) = au + av$。
8.  **标量乘法对标量加法的分配律**：$(a + b)v = av + bv$。
9.  **标量乘法结合律**：$a(bv) = (ab)v$。
10. **乘法单位元存在性**：$1 \cdot v = v$。

#### 向量空间的示例

1.  **欧几里得空间**：$\mathbb{R}^n$，具有标准的加法和标量乘法。
2.  **多项式**：所有实系数多项式的集合，$\mathbb{R}[x]$。
3.  **函数**：区间 $[0,1]$ 上所有连续函数的集合，具有函数加法和标量乘法。
4.  **矩阵**：所有具有实条目的 $m \times n$ 矩阵的集合。
5.  **序列**：所有无限实序列 $(a_1, a_2, \dots)$ 的集合。

所有这些都满足向量空间公理。

#### Python 数值演示：矩阵向量空间

我们可以使用 `numpy` 来验证矩阵集合在加法和数乘下的封闭性及公理行为。

```python
import numpy as np

# 定义两个 2x2 矩阵 (属于向量空间 V)
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

# 1. 验证加法封闭性 (结果仍是 2x2 矩阵)
C = A + B
print("A + B =\n", C)

# 2. 验证标量乘法封闭性
scalar = 3.0
D = scalar * A
print("3 * A =\n", D)

# 3. 验证分配律 a(A + B) = aA + aB
lhs = scalar * (A + B)
rhs = scalar * A + scalar * B
print("分配律成立吗？", np.allclose(lhs, rhs))

# 4. 验证零向量 (零矩阵)
Zero = np.zeros((2, 2))
print("A + 0 = A 吗？", np.allclose(A + Zero, A))
```

#### 非示例（反例）

1.  **自然数集合 $\mathbb{N}$** 不是向量空间（没有加法逆元，例如没有负数）。
2.  **正实数集合 $\mathbb{R}^+$** 不是向量空间（对负标量乘法不封闭，例如 $(-1) \times 2 = -2 \notin \mathbb{R}^+$）。
3.  **恰好 2 次的多项式集合** 不是向量空间（对加法不封闭：$x^2 + x^2 = 2x^2$ 仍是 2 次，但 $x^2 - x^2 = 0$ 是 0 次，不被允许）。

这些例子表明了公理的重要性：如果没有它们，结构就会崩溃。

#### 零向量

每个向量空间必须包含一个零向量。这不是可选项。它是加法的“无为”元素。
*   在 $\mathbb{R}^n$ 中，它是 $(0,0,\dots,0)$。
*   在多项式中，它是零多项式 $p(x) = 0$。
*   在函数空间中，它是函数 $f(x) = 0$。

> **提示**：零向量必须是唯一的。如果一个集合中不存在“零元素”，它就不能构成向量空间。

#### 加法逆元

对于每个向量 $v$，我们需要 $-v$。这确保了像 $u+v=w$ 这样的方程总是可以重排为 $u=w-v$。如果没有加法逆元，求解线性方程将无法进行。

#### 标量与域

标量来自一个**域**：通常是实数 $\mathbb{R}$ 或复数 $\mathbb{C}$。标量的选择很重要：

*   在 $\mathbb{R}$ 上，多项式空间与在 $\mathbb{C}$ 上不同。
*   在有限域上（如模 $p$ 整数），向量空间存在于离散数学和编码理论中。

#### 几何解释

*   公理保证向量可以以可预测的方式相加和缩放。
*   **封闭性**确保空间是“自包含”的。
*   **加法逆元**确保对称性：每个方向都可以反转。
*   **分配律**确保缩放和加法之间的一致性。

Together, these rules make vector spaces stable and reliable mathematical objects.
这些规则共同使得向量空间成为稳定且可靠的数学对象。

#### 日常类比

*   **语言**：单词通过语法规则形成句子。向量空间是加法和缩放遵循严格类似语法规则的集合。
*   **音乐**：音符组合（加法）并改变音高（缩放）。只有当这些操作保持在系统内时，音乐空间才有意义。
*   **建筑**：砖块和砂浆可以组合（加法）和缩放（更大或更小的结构），但只有遵循稳定性规则时才行。

#### 为什么这很重要

1.  向量空间在单一框架下统一了数学的许多领域。
2.  它们将 $\mathbb{R}^n$ 推广到函数、多项式及更多对象。
3.  公理保证线性代数的所有工具—— span（生成）、basis（基）、dimension（维数）、linear maps（线性映射）——都适用。
4.  识别伪装下的向量空间是高级数学和物理中的重要一步。

#### 练习题解答与详细过程

##### 1. 验证 2×2 矩阵集合是向量空间

**解答**：
设 $V$ 为所有 $2 \times 2$ 实矩阵的集合。设 $A, B, C \in V$，标量 $k, l \in \mathbb{R}$。
我们需要验证公理是否成立。由于矩阵运算定义为对应元素的运算，而实数满足域公理，因此矩阵继承了许多性质。

*   **封闭性**：两个 $2 \times 2$ 矩阵相加仍为 $2 \times 2$ 矩阵；标量乘矩阵仍为 $2 \times 2$ 矩阵。
*   **加法性质**：
    *   交换律：$A + B = B + A$（实数加法交换律）。
    *   结合律：$(A + B) + C = A + (B + C)$。
    *   零向量：零矩阵 $\mathbf{0} = \begin{pmatrix} 0 & 0 \\ 0 & 0 \end{pmatrix}$ 满足 $A + \mathbf{0} = A$。
    *   逆元：对于 $A = \begin{pmatrix} a & b \\ c & d \end{pmatrix}$，存在 $-A = \begin{pmatrix} -a & -b \\ -c & -d \end{pmatrix}$ 使得 $A + (-A) = \mathbf{0}$。
*   **标量乘法性质**：
    *   分配律 1：$k(A + B) = kA + kB$。
    *   分配律 2：$(k + l)A = kA + lA$。
    *   结合律：$k(lA) = (kl)A$。
    *   单位元：$1 \cdot A = A$。

**结论**：所有公理均满足，故 $V$ 是向量空间。

##### 2. 多项式集合：次数 $\le 3$ vs 次数 $= 3$

**解答**：
*   **情况 A：次数不超过 3 的多项式集合 $P_3$**
    设 $p(x), q(x) \in P_3$。
    1.  **加法封闭性**：两个次数 $\le 3$ 的多项式相加，最高次项不会超过 3 次。例如 $(x^3 + 1) + (-x^3 + x) = x + 1$，次数为 1，仍 $\le 3$。
    2.  **零向量**：零多项式 $0$ 的次数通常定义为 $-\infty$ 或 undefined，但在 $P_3$ 中它被视为满足次数 $\le 3$ 的条件。
    3.  其他公理（结合律、分配律等）继承自多项式运算性质。
    **结论**：$P_3$ 是向量空间。

*   **情况 B：次数恰好为 3 的多项式集合 $S$**
    设 $p(x) = x^3$, $q(x) = -x^3$。显然 $p, q \in S$。
    1.  **加法封闭性失效**：$p(x) + q(x) = 0$。零多项式的次数不是 3。因此 $p+q \notin S$。
    2.  **零向量缺失**：集合中不包含零多项式（因为零多项式次数不是 3）。
    **结论**：$S$ 不是向量空间。

##### 3. 偶函数集合是否为向量空间

**解答**：
设 $V = \{ f : \mathbb{R} \to \mathbb{R} \mid f(-x) = f(x) \}$ 为所有偶函数的集合。
设 $f, g \in V$，标量 $c \in \mathbb{R}$。

1.  **加法封闭性**：
    考虑 $(f+g)(x) = f(x) + g(x)$。
    检查偶性：$(f+g)(-x) = f(-x) + g(-x) = f(x) + g(x) = (f+g)(x)$。
    所以 $f+g \in V$。
2.  **标量乘法封闭性**：
    考虑 $(cf)(x) = c \cdot f(x)$。
    检查偶性：$(cf)(-x) = c \cdot f(-x) = c \cdot f(x) = (cf)(x)$。
    所以 $cf \in V$。
3.  **零向量**：
    零函数 $z(x) = 0$ 满足 $z(-x) = 0 = z(x)$，是偶函数。
4.  **其他公理**：
    函数加法和数乘满足结合律、分配律等，因为它们基于实数运算。

**结论**：偶函数集合是一个向量空间（它是所有函数空间的子空间）。

##### 4. 挑战：可微函数集合

**解答**：
设 $V = \{ f : [0,1] \to \mathbb{R} \mid f \text{ 是可微的} \}$。

1.  **加法封闭性**：
    微分法则表明，如果 $f$ 和 $g$ 可微，则 $(f+g)' = f' + g'$ 存在。
    因此 $f+g$ 也是可微的。
2.  **标量乘法封闭性**：
    如果 $f$ 可微，则 $(cf)' = c f'$ 存在。
    因此 $cf$ 也是可微的。
3.  **零向量**：
    零函数 $f(x)=0$ 是可微的（导数为 0）。
4.  **逆元**：
    如果 $f$ 可微，$-f$ 也可微。
5.  **其他公理**：
    函数的加法和数乘天然满足向量空间的代数公理（交换律、结合律、分配律）。

**结论**：$[0,1]$ 上所有可微函数的集合构成一个向量空间。这在泛函分析中是非常基础的空间之一。
