### 1. 标量、向量与坐标系

当我们开始学习线性代数时，一切始于最基础的构建模块：**标量**（Scalars）和**向量**（Vectors）。

**标量**只是一个单独的数字，例如 $3$、$-7$ 或 $\pi$。它只携带**大小**（magnitude）信息，而没有方向。标量是我们用来计数、测量长度或将其他对象进行缩放的工具。

相比之下，**向量**是一组有序的数字集合。你可以将其想象为空间中指向某个位置的箭头，或者简单地看作一个列表，例如 2D 空间中的 $(2, 5)$ 或 3D 空间中的 $(1, -3, 4)$。如果说标量衡量的是“有多少”，那么向量衡量的则是“有多少”以及“朝哪个方向”。

#### 坐标系

为了讨论向量，我们需要一个**坐标系**。想象在一张纸上画出两条相互垂直的轴：$x$ 轴（从左到右）和 $y$ 轴（从下到上）。纸上的每一个点都可以用两个数字来描述：沿 $x$ 轴的距离和沿 $y$ 轴的距离。这对数字构成了一个 2D 向量。

如果再添加一个从纸面垂直向上的 $z$ 轴，你就拥有了 3D 空间。每个坐标系都为我们提供了一种用数值描述向量的方法，尽管底层的“空间”本质是相同的。

> **注意**：坐标系的选择是人为的。同一个向量在不同的坐标系下可能有不同的数值表示，但其几何意义（长度和方向）保持不变。

#### 标量与向量的可视化

- **标量**就像尺子上的单个刻度标记。
- **向量**就像一支从原点 $(0, 0, \dots)$ 出发，指向由其分量定义的终点的箭头。
  例如，2D 中的向量 $(3, 4)$ 表示从原点出发，沿 $x$ 轴移动 3 个单位，再沿 $y$ 轴移动 4 个单位所指向的点。

#### 为什么要从这里开始？

理解标量和向量之间的区别是线性代数中所有后续内容的基石。每一个概念——矩阵、线性变换、特征值——最终都归结为我们如何操作向量以及如何用标量对其进行缩放。如果没有这种区分，学科的其余部分将失去锚点。

#### 为什么这很重要？

几乎每一个科学和工程领域都依赖于这一概念：
- **物理学**使用向量来表示速度、加速度和力。
- **计算机图形学**使用它们来表示点、颜色和变换。
- **数据科学**将整个数据集视为高维向量。

通过尽早掌握标量和向量，你就解锁了现代科学和技术所使用的语言。

#### 练习参考答案与解析

为了帮助你验证自己的理解，以下是上述练习的详细解答过程，并附带 Python 代码示例来直观展示结果。

**1. 绘制向量 $(2, 3)$**
- **解析**：从原点 $(0,0)$ 出发，向右移动 2 个单位，向上移动 3 个单位。箭头终点落在第一象限。

**2. 绘制向量 $(-1, 4)$ 并比较**
- **解析**：从原点 $(0,0)$ 出发，向左移动 1 个单位，向上移动 4 个单位。箭头终点落在第二象限。
- **方向比较**：$(2, 3)$ 指向右上方，$(-1, 4)$ 指向左上方。
- **长度计算**：向量的长度（模）可以通过勾股定理计算：$||\mathbf{v}|| = \sqrt{x^2 + y^2}$。
  - 向量 $(2, 3)$ 的长度：$\sqrt{2^2 + 3^2} = \sqrt{4 + 9} = \sqrt{13} \approx 3.61$
  - 向量 $(-1, 4)$ 的长度：$\sqrt{(-1)^2 + 4^2} = \sqrt{1 + 16} = \sqrt{17} \approx 4.12$

**3. 思考题解答**
- **哪个更向上？** 向量 $(-1, 4)$ 的 $y$ 分量是 $4$，而 $(2, 3)$ 的 $y$ 分量是 $3$。因此，$(-1, 4)$ 指向“更上方”。
- **哪个更长？** 根据上面的计算，$\sqrt{17} > \sqrt{13}$，所以向量 $(-1, 4)$ 更长。

#### Python 代码演示

我们可以使用 `numpy` 来定义向量，并使用 `matplotlib` 来可视化这些向量，验证上述结论。

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义向量
v1 = np.array([2, 3])
v2 = np.array([-1, 4])

# 计算向量的长度 (模)
length_v1 = np.linalg.norm(v1)
length_v2 = np.linalg.norm(v2)

print(f"向量 v1 (2, 3) 的长度：{length_v1:.4f}")
print(f"向量 v2 (-1, 4) 的长度：{length_v2:.4f}")

# 可视化向量
plt.figure(figsize=(6, 6))
origin = np.array([0, 0])

# 绘制向量 v1 (蓝色)
plt.quiver(*origin, *v1, color='blue', scale=1, scale_units='xy', angles='xy', label='v1 (2, 3)')
# 绘制向量 v2 (红色)
plt.quiver(*origin, *v2, color='red', scale=1, scale_units='xy', angles='xy', label='v2 (-1, 4)')

# 设置坐标轴范围
plt.xlim(-5, 5)
plt.ylim(-5, 5)
plt.axhline(0, color='black', linewidth=1) # x 轴
plt.axvline(0, color='black', linewidth=1) # y 轴
plt.grid(True, which='both', linestyle='--', alpha=0.7)
plt.legend()
plt.title('向量可视化：(2, 3) vs (-1, 4)')
plt.xlabel('x 轴')
plt.ylabel('y 轴')

# 保持纵横比相等，确保角度直观
plt.gca().set_aspect('equal')

plt.show()
```

> **提示**：运行上述代码时，请确保你的环境中已安装 `numpy` 和 `matplotlib` 库。通过观察生成的图像，你可以直观地看到红色箭头（$v2$）确实比蓝色箭头（$v1$）更长，且 $y$ 轴方向的分量更大。
