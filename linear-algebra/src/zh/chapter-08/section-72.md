### 72. 正交性与标准正交基

正交性是线性代数中最核心且强大的概念之一。它将欧几里得空间中我们熟悉的“垂直”概念，推广到了装备了内积的抽象向量空间中。当正交性与归一化（使向量长度为 1）相结合时，我们就得到了**标准正交基（Orthonormal Bases）**。这种基能够简化计算、澄清几何结构，并且是许多现代算法的基石。

#### 正交性

两个向量 $x, y \in V$ 被称为**正交（Orthogonal）**，如果它们的内积为零：

$$
\langle x, y \rangle = 0.
$$

- 在 $\mathbb{R}^2$ 或 $\mathbb{R}^3$ 中，这意味着向量在几何上是**垂直**的。
- 在函数空间中，这意味着它们乘积的积分为零。
- 在信号处理中，这意味着信号是独立的且互不重叠。

正交性捕捉了在内积几何意义下“无重叠”或“独立”的核心思想。

> **注意**：零向量与任何向量都正交，但在讨论正交基时，我们通常只考虑非零向量。

#### 正交向量的性质

1. 若 $x \perp y$，则 $\|x+y\|^2 = \|x\|^2 + \|y\|^2$（勾股定理的推广）。
2. 正交性是对称的：若 $x \perp y$，则 $y \perp x$。
3. **任何一组两两正交的非零向量自动线性无关。**

最后一条性质至关重要：**正交性保证了线性无关性**。这意味着我们不需要通过复杂的高斯消元法来验证独立性，只需检查内积即可。

#### 标准正交集

一个**标准正交集（Orthonormal Set）**是一组向量 $\{u_1, \dots, u_k\}$，满足以下条件：

$$
\langle u_i, u_j \rangle = \begin{cases} 
1 & \text{if } i=j, \\ 
0 & \text{if } i \neq j.  
\end{cases}
$$

- 每个向量的长度均为 1（单位向量）。
- 不同的向量之间两两正交。

这种结构使得坐标计算变得尽可能简单。

#### 标准正交基

如果向量空间的一组基 $\{u_1, \dots, u_n\}$ 作为集合是标准正交的，则称其为**标准正交基**。

- 空间中的任意向量 $x \in V$ 都可以表示为：

  $$
  x = \sum_{i=1}^n \langle x, u_i \rangle u_i.
  $$
- 系数仅仅是内积 $\langle x, u_i \rangle$，**无需求解线性方程组**。

这就是标准正交基如此方便的原因：它们使得向量的表示和投影变得毫不费力。

> **提示**：系数 $c_i = \langle x, u_i \rangle$ 通常被称为**傅里叶系数**。这在信号处理中非常常见。

#### Python 数值演示

我们可以使用 `numpy` 来验证标准正交基的便利性。以下代码展示了如何在标准正交基下通过内积直接获取坐标，而不需要矩阵求逆。

```python
import numpy as np

# 定义一个标准正交基 (例如旋转后的基)
# u1 和 u2 是单位向量且互相垂直
u1 = np.array([1/np.sqrt(2), 1/np.sqrt(2)])
u2 = np.array([-1/np.sqrt(2), 1/np.sqrt(2)])
basis = np.column_stack((u1, u2))

# 定义任意向量 x
x = np.array([3, 4])

# 方法 1: 利用标准正交基性质，系数即为内积
c1 = np.dot(x, u1)
c2 = np.dot(x, u2)
coeffs_direct = np.array([c1, c2])

# 方法 2: 普通基需要求解线性方程组 Ax = b (即 x = U * c => c = U^-1 * x)
# 对于正交矩阵，逆等于转置，但这里演示通用逻辑对比
coeffs_solve = np.linalg.solve(basis, x)

print("直接内积计算系数:", coeffs_direct)
print("求解方程组计算系数:", coeffs_solve)
print("两者是否接近:", np.allclose(coeffs_direct, coeffs_solve))

# 验证重构
x_reconstructed = c1 * u1 + c2 * u2
print("重构误差:", np.linalg.norm(x - x_reconstructed))
```

#### 示例

1. **$\mathbb{R}^n$ 中的标准基**：
   $\{e_1, e_2, \dots, e_n\}$，其中 $e_i$ 在第 $i$ 个坐标为 1，其余为 0。
   - 在标准点积下是标准正交的。

2. **傅里叶基**：
   区间 $[0,2\pi]$ 上的函数 $\{\sin(nx), \cos(nx)\}$ 在内积
   $\langle f,g\rangle = \int_0^{2\pi} f(x)g(x)dx$ 下是正交的。
   - 该基将信号分解为纯频率成分。

3. **多项式基**：
   勒让德多项式 $P_n(x)$ 在 $[-1,1]$ 上关于内积
   $\langle f,g\rangle = \int_{-1}^1 f(x)g(x)\,dx$ 正交。

#### 几何意义

正交性将空间分割为独立的“方向”。

- 标准正交基就像完美对齐的坐标轴。
- 任何向量都可以唯一地分解为沿这些轴的独立贡献之和。
- 距离和角度得以保持，使得几何结构透明清晰。

#### 生活中的类比

- **声音频率**：如果音符是正交的（独立频率），和弦中的每个音符都可以被分离出来。
- **数据分析**：正交轴对应独立的特征（无相关性）。
- **体育**：比赛中的进攻和防守是正交的角色——不同但都必要。
- **金融**：正交的投资组合没有相关性——一个组合的风险不会影响另一个。

#### 应用领域

1. **信号处理**：将信号分解为正交的频率分量。
2. **机器学习**：主成分分析（PCA）形成标准正交基，捕捉方差方向。
3. **数值方法**：标准正交基提高了数值稳定性（避免病态矩阵）。
4. **量子力学**：如果状态代表互斥的结果，则它们是正交的。
5. **计算机图形学**：旋转由具有标准正交列的正交矩阵表示。

#### 为什么这很重要

- 正交性提供独立性；标准正交性提供归一化。
- 两者结合使得计算、分解和投影变得干净且高效。
- 它们是傅里叶分析、主成分分析以及无数现代算法的基础。

### 练习题详细解答

#### 1. 证明 $\{(1,0,0), (0,1,0), (0,0,1)\}$ 是 $\mathbb{R}^3$ 的标准正交基

**解答过程：**
令 $u_1 = (1,0,0)$, $u_2 = (0,1,0)$, $u_3 = (0,0,1)$。
我们需要验证两个条件：
1.  **单位长度**：
    - $\|u_1\| = \sqrt{1^2+0^2+0^2} = 1$
    - $\|u_2\| = \sqrt{0^2+1^2+0^2} = 1$
    - $\|u_3\| = \sqrt{0^2+0^2+1^2} = 1$
    所有向量长度均为 1。
2.  **两两正交**：
    - $\langle u_1, u_2 \rangle = 1\cdot0 + 0\cdot1 + 0\cdot0 = 0$
    - $\langle u_1, u_3 \rangle = 1\cdot0 + 0\cdot0 + 0\cdot1 = 0$
    - $\langle u_2, u_3 \rangle = 0\cdot0 + 1\cdot0 + 0\cdot1 = 0$
    所有不同向量的内积均为 0。

**结论**：该集合是标准正交集。由于它包含 3 个向量且位于 $\mathbb{R}^3$ 中，且线性无关（正交非零向量必无关），因此它构成 $\mathbb{R}^3$ 的标准正交基。

---

#### 2. 检查 $\{(1,1,0), (1,-1,0), (0,0,1)\}$ 是否标准正交，若否则标准化

**解答过程：**
令 $v_1 = (1,1,0)$, $v_2 = (1,-1,0)$, $v_3 = (0,0,1)$。

1.  **检查正交性**：
    - $\langle v_1, v_2 \rangle = 1\cdot1 + 1\cdot(-1) + 0\cdot0 = 1 - 1 = 0$ (正交)
    - $\langle v_1, v_3 \rangle = 1\cdot0 + 1\cdot0 + 0\cdot1 = 0$ (正交)
    - $\langle v_2, v_3 \rangle = 1\cdot0 + (-1)\cdot0 + 0\cdot1 = 0$ (正交)
    向量两两正交。

2.  **检查长度（归一化）**：
    - $\|v_1\| = \sqrt{1^2+1^2+0^2} = \sqrt{2} \neq 1$
    - $\|v_2\| = \sqrt{1^2+(-1)^2+0^2} = \sqrt{2} \neq 1$
    - $\|v_3\| = \sqrt{0^2+0^2+1^2} = 1$
    长度不全为 1，因此**不是**标准正交集。

3.  **标准化**：
    将每个向量除以其长度：
    - $u_1 = \frac{v_1}{\|v_1\|} = \left(\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}, 0\right)$
    - $u_2 = \frac{v_2}{\|v_2\|} = \left(\frac{1}{\sqrt{2}}, -\frac{1}{\sqrt{2}}, 0\right)$
    - $u_3 = \frac{v_3}{\|v_3\|} = (0, 0, 1)$

**结论**：原集合正交但不标准。标准化后的标准正交基为 $\left\{\left(\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}, 0\right), \left(\frac{1}{\sqrt{2}}, -\frac{1}{\sqrt{2}}, 0\right), (0, 0, 1)\right\}$。

#### 3. 计算 $x=(3,4)$ 在不同基下的系数

**解答过程：**

**情况 A：标准基 $e_1=(1,0), e_2=(0,1)$**
由于是标准正交基，系数 $c_i = \langle x, e_i \rangle$。
- $c_1 = 3\cdot1 + 4\cdot0 = 3$
- $c_2 = 3\cdot0 + 4\cdot1 = 4$
系数为 $(3, 4)$。这很直观，因为这就是向量的坐标。

**情况 B：旋转后的标准正交基**
令 $u_1 = (1/\sqrt{2}, 1/\sqrt{2})$, $u_2 = (-1/\sqrt{2}, 1/\sqrt{2})$。
首先验证它们是标准正交的（略，同第 2 题逻辑）。
计算系数 $c_i = \langle x, u_i \rangle$：
- $c_1 = \langle (3,4), (1/\sqrt{2}, 1/\sqrt{2}) \rangle = 3\cdot\frac{1}{\sqrt{2}} + 4\cdot\frac{1}{\sqrt{2}} = \frac{7}{\sqrt{2}} = \frac{7\sqrt{2}}{2}$
- $c_2 = \langle (3,4), (-1/\sqrt{2}, 1/\sqrt{2}) \rangle = 3\cdot(-\frac{1}{\sqrt{2}}) + 4\cdot\frac{1}{\sqrt{2}} = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$

**结论**：
- 在标准基下系数为 $(3, 4)$。
- 在旋转基下系数为 $(\frac{7\sqrt{2}}{2}, \frac{\sqrt{2}}{2})$。
这展示了基变换如何改变坐标表示，但向量本身的几何位置不变。

#### 4. 挑战：证明有限维内积空间中标准正交基总是存在

**解答过程：**

**证明思路（基于格拉姆 - 施密特正交化 Gram-Schmidt Process）：**

1.  **存在基**：任何有限维向量空间 $V$ (设维数为 $n$) 都存在一组基 $\{v_1, v_2, \dots, v_n\}$。
2.  **正交化过程**：我们可以通过归纳法构造一组正交基 $\{w_1, w_2, \dots, w_n\}$。
    - 令 $w_1 = v_1$。
    - 令 $w_2 = v_2 - \text{proj}_{w_1}(v_2) = v_2 - \frac{\langle v_2, w_1 \rangle}{\langle w_1, w_1 \rangle}w_1$。此时 $w_2 \perp w_1$。
    - 一般地，令 $w_k = v_k - \sum_{j=1}^{k-1} \frac{\langle v_k, w_j \rangle}{\langle w_j, w_j \rangle}w_j$。
    - 通过构造，$w_k$ 与之前所有的 $w_1, \dots, w_{k-1}$ 正交。
3.  **非零性**：由于 $\{v_i\}$ 线性无关，构造出的 $\{w_i\}$ 均为非零向量。
4.  **归一化**：令 $u_i = \frac{w_i}{\|w_i\|}$。
5.  **结论**：集合 $\{u_1, \dots, u_n\}$ 是一组标准正交基。

**证毕**。这个过程不仅证明了存在性，还提供了构造标准正交基的具体算法。
