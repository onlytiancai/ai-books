### 10. $\mathbb{R}^2$ 和 $\mathbb{R}^3$ 中的标准正交组

到目前为止，我们已经讨论了向量、它们的长度、夹角以及如何将一个向量投影到另一个向量上。这些概念的自然 culmination（ culmination 意为 culmination/顶点/集大成者，此处指逻辑发展的必然结果）便是**标准正交组**（Orthonormal Sets）的概念。这是一组不仅相互正交（mutually perpendicular，即互相垂直），而且已经标准化（normalized，即长度均为 1）的向量集合。标准正交组构成了线性代数中最简洁、最高效的坐标系。这就好比你在数学上拥有了一套互相垂直且刻度精确为单位长度的尺子。

#### 正交与标准化

让我们把“标准正交”（orthonormal）这个词拆解为两部分来理解：

- **正交（Orthogonal）**：如果两个向量 $u$ 和 $v$ 满足 $u \cdot v = 0$，则称它们正交。
  - 在 $\mathbb{R}^2$ 中，这意味着两个向量相交成直角。
  - 在 $\mathbb{R}^3$ 中，这意味着它们指向互相垂直的方向。

- **标准化（Normalized）**：如果向量 $v$ 的长度为 $1$，即 $\|v\| = 1$，则称它是标准化的。
  - 这样的向量被称为**单位向量**（unit vectors）。

当我们同时满足这两个条件时，就得到了**标准正交向量**：它们既互相垂直，长度又为单位长度。

> **注意**：正交向量不一定长度为 1，但标准正交向量必须长度为 1。标准化过程通常通过将向量除以其长度来实现：$u_{normalized} = \frac{u}{\|u\|}$。

#### $\mathbb{R}^2$ 中的标准正交组

在二维空间中，一个标准正交组通常由两个向量组成。
最经典的例子是标准基：

$e_1 = (1, 0), \quad e_2 = (0, 1)$

- 点积：$e_1 \cdot e_2 = (1 \times 0 + 0 \times 1) = 0 \;\;\Rightarrow\;\;$ 正交
- 长度：$\|e_1\| = 1$, $\|e_2\| = 1 \;\;\Rightarrow\;\;$ 标准化

因此，$\{e_1, e_2\}$ 是一个标准正交组。
事实上，这是 $\mathbb{R}^2$ 的**标准基**（standard basis）。
任何向量 $(x, y)$ 都可以写成 $x e_1 + y e_2$。
这是最简单的坐标系统。

我们可以用 Python 来验证这一性质：

```python
import numpy as np

# 定义标准基向量
e1 = np.array([1, 0])
e2 = np.array([0, 1])

# 验证正交性 (点积应为 0)
dot_product = np.dot(e1, e2)
print(f"e1 · e2 = {dot_product}")

# 验证标准化 (长度应为 1)
norm_e1 = np.linalg.norm(e1)
norm_e2 = np.linalg.norm(e2)
print(f"||e1|| = {norm_e1}, ||e2|| = {norm_e2}")
```

#### $\mathbb{R}^3$ 中的标准正交组

在三维空间中，一个标准正交组通常包含三个向量。
其标准基为：

$e_1 = (1, 0, 0), \quad e_2 = (0, 1, 0), \quad e_3 = (0, 0, 1)$

- 每一对向量的点积均为零，因此它们相互正交
- 每个向量的长度均为 $1$，因此它们是标准化的
- 它们共同张成（span）了整个 $\mathbb{R}^3$ 空间

几何上，它们对应于三维空间中的 $x$ 轴、$y$ 轴和 $z$ 轴。
任何向量 $(x, y, z)$ 都可以写成线性组合 $x e_1 + y e_2 + z e_3$。

#### 超越标准基

标准基并不是唯一的标准正交组。例如，考虑以下向量：

$u = \left(\tfrac{1}{\sqrt{2}}, \tfrac{1}{\sqrt{2}}\right), \quad 
 v = \left(-\tfrac{1}{\sqrt{2}}, \tfrac{1}{\sqrt{2}}\right)$

- 点积：$(\tfrac{1}{\sqrt{2}})(-\tfrac{1}{\sqrt{2}}) + (\tfrac{1}{\sqrt{2}})(\tfrac{1}{\sqrt{2}}) = -\tfrac{1}{2} + \tfrac{1}{2} = 0$
- 长度：$\sqrt{(\tfrac{1}{\sqrt{2}})^2 + (\tfrac{1}{\sqrt{2}})^2} = \sqrt{\tfrac{1}{2} + \tfrac{1}{2}} = 1$

所以 $\{u, v\}$ 在 $\mathbb{R}^2$ 中也是标准正交的。
这些向量相对于标准轴旋转了 $45^\circ$。

同样地，在 $\mathbb{R}^3$ 中，你也可以构造旋转后的标准正交组（例如沿对角线的单位向量），只要满足垂直性和单位长度的条件即可。

下面我们用代码验证这组旋转后的向量：

```python
import numpy as np

# 定义旋转后的向量
u = np.array([1/np.sqrt(2), 1/np.sqrt(2)])
v = np.array([-1/np.sqrt(2), 1/np.sqrt(2)])

# 验证正交性
print(f"u · v = {np.dot(u, v):.10f}")  # 应接近 0

# 验证长度
print(f"||u|| = {np.linalg.norm(u):.10f}")  # 应接近 1
print(f"||v|| = {np.linalg.norm(v):.10f}")  # 应接近 1
```

#### 标准正交组的性质

1. **简化的坐标计算**：如果 $\{v_1, \ldots, v_k\}$ 是一个标准正交组，那么对于其张成空间中的任意向量 $u$，其坐标系数非常容易计算：
   $$
   c_i = u \cdot v_i
   $$
   这比求解线性方程组要简单得多。

2. **勾股定理的推广**：如果向量是标准正交的，它们之和的长度平方等于系数平方之和。
   例如，如果 $u = a v_1 + b v_2$，那么
   $$
   \|u\|^2 = a^2 + b^2
   $$

3. **投影变得简单**：投影到标准正交组上非常直接——只需计算点积。

4. **矩阵性质优良**：当这些向量构成矩阵的列时，正交性使得该矩阵成为**正交矩阵**（Orthogonal Matrix），它具有特殊性质：其转置等于其逆矩阵，且保持长度和角度不变。

> **提示**：性质 1 是傅里叶级数系数的核心思想。在标准正交基下，寻找分量就像是在做投影，不需要解方程组。

#### 在 $\mathbb{R}^2$ 和 $\mathbb{R}^3$ 中的重要性

- **几何学**：标准正交基对应于坐标轴。
- **物理学**：它们代表独立的运动方向或力的方向。
- **计算机图形学**：标准正交组定义了相机轴和物体旋转。
- **工程学**：它们简化了应力、应变和旋转分析。

尽管 $\mathbb{R}^2$ 和 $\mathbb{R}^3$ 相对简单，但相同的思想自然地扩展到了更高维度，在那里虽然无法可视化，但代数运算完全相同。

#### 日常类比

- **城市导航**：想象两条垂直的街道，每条都以米为单位标记。向东走 3 个单位，向北走 4 个单位，得到坐标 (3, 4)。这就是一个标准正交系统。
- **家具组装**：当你被指示将一个部件“垂直向上”对齐，另一个“水平横跨”对齐时，你正在使用标准正交方向。
- **数字屏幕**：计算机显示器使用水平和垂直对齐的像素网格，这是标准正交性的实际实现。

#### 为什么标准正交组很重要

标准正交性是线性代数中构建基的“黄金标准”：

- 它使计算快速且简单。
- 它确保了计算中的数值稳定性（这在算法和模拟中至关重要）。
- 它是关键分解方法的基础，如 QR 分解、奇异值分解（SVD）和谱定理。
- 它提供了思考空间的最清晰方式：正交、独立的方向，缩放为单位长度。

只要可能，数学家和工程师更喜欢标准正交基而不是任意基。

#### 参考答案与详细解析

**1. 验证 $(3/5, 4/5)$ 和 $(–4/5, 3/5)$ 是否标准正交**

**解答过程：**
设 $v_1 = (3/5, 4/5)$，$v_2 = (-4/5, 3/5)$。

*   **步骤 1：验证正交性**
    计算点积 $v_1 \cdot v_2$：
    $$
    v_1 \cdot v_2 = \left(\frac{3}{5}\right)\left(-\frac{4}{5}\right) + \left(\frac{4}{5}\right)\left(\frac{3}{5}\right) = -\frac{12}{25} + \frac{12}{25} = 0
    $$
    点积为 0，说明两者正交。

*   **步骤 2：验证标准化（单位长度）**
    计算 $v_1$ 的长度：
    $$
    \|v_1\| = \sqrt{\left(\frac{3}{5}\right)^2 + \left(\frac{4}{5}\right)^2} = \sqrt{\frac{9}{25} + \frac{16}{25}} = \sqrt{\frac{25}{25}} = 1
    $$
    计算 $v_2$ 的长度：
    $$
    \|v_2\| = \sqrt{\left(-\frac{4}{5}\right)^2 + \left(\frac{3}{5}\right)^2} = \sqrt{\frac{16}{25} + \frac{9}{25}} = \sqrt{1} = 1
    $$
    长度均为 1，说明已标准化。

**结论**：$\{v_1, v_2\}$ 是标准正交组。

**Python 验证代码：**
```python
import numpy as np

v1 = np.array([3/5, 4/5])
v2 = np.array([-4/5, 3/5])

print(f"正交性检查 (应为 0): {np.dot(v1, v2)}")
print(f"v1 长度 (应为 1): {np.linalg.norm(v1)}")
print(f"v2 长度 (应为 1): {np.linalg.norm(v2)}")
```

**2. 构造 $\mathbb{R}^3$ 中的非标准基标准正交组**

**解答过程：**
我们需要三个互相垂直且长度为 1 的向量。
*   **步骤 1：选择第一个向量**
    根据提示，设 $u_1 = (1/\sqrt{2}, 1/\sqrt{2}, 0)$。
    验证长度：$\sqrt{1/2 + 1/2 + 0} = 1$。

*   **步骤 2：构造第二个向量**
    我们需要 $u_2$ 与 $u_1$ 正交。观察 $u_1$ 的前两个分量，我们可以交换并取反其中一个来构造垂直向量，保持 z 分量为 0。
    设 $u_2 = (-1/\sqrt{2}, 1/\sqrt{2}, 0)$。
    验证正交：$u_1 \cdot u_2 = -1/2 + 1/2 + 0 = 0$。
    验证长度：$\sqrt{1/2 + 1/2} = 1$。

*   **步骤 3：构造第三个向量**
    $u_3$ 必须与前两个向量都正交。由于 $u_1$ 和 $u_2$ 的 $z$ 分量均为 0，它们位于 $xy$ 平面内。因此，$z$ 轴方向的单位向量自然与它们垂直。
    设 $u_3 = (0, 0, 1)$。
    验证正交：$u_1 \cdot u_3 = 0$, $u_2 \cdot u_3 = 0$。
    验证长度：$\|u_3\| = 1$。

**结论**：$\{u_1, u_2, u_3\}$ 是一组满足条件的标准正交向量。

**3. 计算 $u = (2, 1)$ 在给定基下的坐标**

**解答过程：**
给定标准正交基 $v_1 = (1/\sqrt{2}, 1/\sqrt{2})$ 和 $v_2 = (-1/\sqrt{2}, 1/\sqrt{2})$。
我们要找系数 $c_1, c_2$ 使得 $u = c_1 v_1 + c_2 v_2$。
利用标准正交基的性质 $c_i = u \cdot v_i$：

*   **计算 $c_1$**：
    $$
    c_1 = u \cdot v_1 = 2 \cdot \frac{1}{\sqrt{2}} + 1 \cdot \frac{1}{\sqrt{2}} = \frac{3}{\sqrt{2}} = \frac{3\sqrt{2}}{2}
    $$

*   **计算 $c_2$**：
    $$
    c_2 = u \cdot v_2 = 2 \cdot \left(-\frac{1}{\sqrt{2}}\right) + 1 \cdot \frac{1}{\sqrt{2}} = -\frac{2}{\sqrt{2}} + \frac{1}{\sqrt{2}} = -\frac{1}{\sqrt{2}} = -\frac{\sqrt{2}}{2}
    $$

**结论**：$u$ 在该基下的坐标为 $(\frac{3\sqrt{2}}{2}, -\frac{\sqrt{2}}{2})$。

**Python 验证代码：**
```python
import numpy as np

u = np.array([2, 1])
v1 = np.array([1/np.sqrt(2), 1/np.sqrt(2)])
v2 = np.array([-1/np.sqrt(2), 1/np.sqrt(2)])

c1 = np.dot(u, v1)
c2 = np.dot(u, v2)

print(f"坐标 c1: {c1}")
print(f"坐标 c2: {c2}")

# 验证重构
u_reconstructed = c1 * v1 + c2 * v2
print(f"重构向量: {u_reconstructed}")
```

**4. 证明：若 $\{v_1, \ldots, v_k\}$ 标准正交，则 $Q^T Q = I$**

**解答过程：**
设矩阵 $Q$ 的列向量为 $v_1, \ldots, v_k$，即 $Q = [v_1 \mid v_2 \mid \cdots \mid v_k]$。
我们需要计算 $Q^T Q$。

根据矩阵乘法的定义，$(Q^T Q)_{ij}$ 表示 $Q^T$ 的第 $i$ 行与 $Q$ 的第 $j$ 列的点积。
$Q^T$ 的第 $i$ 行其实就是 $Q$ 的第 $i$ 列的转置，即 $v_i^T$。
$Q$ 的第 $j$ 列是 $v_j$。

因此，$(Q^T Q)_{ij} = v_i
