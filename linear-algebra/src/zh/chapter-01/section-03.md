### 3. 向量加法与标量乘法

一旦我们学会了如何用分量和箭头来描述向量，下一步就是学习如何将它们组合起来。线性代数有两个基石般的运算：**向量加法**和**标量乘法**（即用数字缩放向量）。这两个操作虽然简单，却是构建后续所有知识的 foundation。利用它们，我们可以描述运动、力、数据变换以及更多实际应用场景。

#### 坐标中的向量加法

假设我们在二维空间中有两个向量：  
$\mathbf{u} = (u_1, u_2), \quad \mathbf{v} = (v_1, v_2)$。

它们的和定义为：  
$$
\mathbf{u} + \mathbf{v} = (u_1 + v_1, \; u_2 + v_2).
$$

换句话说，就是将**对应的分量相加**。  
这一规则同样适用于更高维度的空间：  

$$
(u_1, u_2, \ldots, u_n) + (v_1, v_2, \ldots, v_n) = (u_1 + v_1, \; u_2 + v_2, \; \ldots, \; u_n + v_n).
$$

示例：
$$
(2, 3) + (-1, 4) = (2 - 1, \; 3 + 4) = (1, 7).
$$

> **注意**：只有当两个向量的维度相同时，才能进行加法运算。你不能将一个二维向量和一个三维向量相加。

我们可以使用 Python 的 `numpy` 库来验证这一计算：

```python
import numpy as np

# 定义向量 u 和 v
u = np.array([2, 3])
v = np.array([-1, 4])

# 向量加法
result = u + v

print(f"u = {u}")
print(f"v = {v}")
print(f"u + v = {result}")
# 输出：u + v = [1 7]
```

#### 向量加法的几何意义

几何直观往往更能揭示本质。如果你将向量 $\mathbf{u}$ 画成一个箭头，然后将向量 $\mathbf{v}$ 的**尾部**放置在 $\mathbf{u}$ 的**头部**，那么从 $\mathbf{u}$ 的起点指向 $\mathbf{v}$ 的终点的箭头就是 $\mathbf{u} + \mathbf{v}$。这被称为**首尾相接法则**（Tip-to-tail rule）。

另一种可视化方法是**平行四边形法则**：将 $\mathbf{u}$ 和 $\mathbf{v}$ 的尾部放在一起（通常放在原点），以它们为邻边形成一个平行四边形，其对角线即为它们的和。

示例：
$\mathbf{u} = (3, 1), \mathbf{v} = (2, 2)$。从原点画出两者。它们的和 $(5, 3)$ 恰好是它们所张成的平行四边形的对角线。

> **提示**：在物理中，这对应于力的合成。如果你同时受到两个力的作用，合力就是这两个力向量的和。

#### 坐标中的标量乘法

标量可以拉伸或收缩向量。  
如果 $\mathbf{u} = (u_1, u_2, \ldots, u_n)$ 且 $c$ 是一个标量，则：  

$$
c \cdot \mathbf{u} = (c \cdot u_1, \; c \cdot u_2, \; \ldots, \; c \cdot u_n).
$$

示例：

$$
2 \cdot (3, 4) = (6, 8).
$$  

$$
(-1) \cdot (3, 4) = (-3, -4).
$$

乘以正标量会保持方向不变，同时拉伸或压缩箭头的长度。  
乘以负标量会将箭头翻转到相反的方向。

使用 Python 演示标量乘法：

```python
# 定义向量 u
u = np.array([3, 4])

# 标量乘法
scaled_2 = 2 * u
scaled_neg1 = -1 * u

print(f"2 * u = {scaled_2}")
print(f"-1 * u = {scaled_neg1}")
# 输出：2 * u = [6 8]
# 输出：-1 * u = [-3 -4]
```

#### 标量乘法的几何意义

想象向量 $(1, 2)$。在坐标纸上画出它：向右 1 格，向上 2 格。现在将其加倍：$(2, 4)$。箭头指向相同的方向，但长度变为原来的两倍。将其减半：$(0.5, 1)$。方向相同但更短。将其取反：$(–1, –2)$。现在箭头指向完全相反的方向。

这一几何图像解释了为什么我们称这些数为“标量”（Scalar）：它们用来**缩放**（Scale）向量。

#### 两者结合：线性组合

向量加法和标量乘法不仅仅是独立的技巧——它们结合形成了线性代数的核心：**线性组合**（Linear Combinations）。

向量 $\mathbf{u}$ 和 $\mathbf{v}$ 的线性组合是指任何形式为  
$a \cdot \mathbf{u} + b \cdot \mathbf{v}$ 的向量，其中 $a$ 和 $b$ 是标量。

示例：  
如果 $\mathbf{u} = (1, 0)$ 和 $\mathbf{v} = (0, 1)$，则  
$3 \cdot \mathbf{u} + 2 \cdot \mathbf{v} = (3, 2)$。

这表明网格上的任何点都可以通过缩放和相加这两个基本向量来到达。  
这就是构建空间的本质。

```python
# 线性组合示例
u = np.array([1, 0])
v = np.array([0, 1])
a, b = 3, 2

linear_combination = a * u + b * v
print(f"3*u + 2*v = {linear_combination}")
# 输出：3*u + 2*v = [3 2]
```

> **注意**：线性组合的概念至关重要。后续我们将学习“张成空间”（Span）和“基”（Basis），它们都是基于线性组合定义的。

#### 代数性质

向量加法和标量乘法遵循与数字算术相似的规则：

- 交换律：$\mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}$  
- 结合律：$(\mathbf{u} + \mathbf{v}) + \mathbf{w} = \mathbf{u} + (\mathbf{v} + \mathbf{w})$  
- 对标量的分配律：$c \cdot (\mathbf{u} + \mathbf{v}) = c \cdot \mathbf{u} + c \cdot \mathbf{v}$  
- 对数字的分配律：$(a + b) \cdot \mathbf{u} = a \cdot \mathbf{u} + b \cdot \mathbf{u}$

这些规则不仅仅是琐碎的记录——它们保证了线性代数的行为是可预测的，这也是它能成为科学语言的原因。

#### 日常类比

- **行走方向**：如果你向北走 3 步，然后向东走 4 步，这就像相加 $(0, 3) + (4, 0) = (4, 3)$。
- **物理中的力**：如果两个人朝不同方向推一个箱子，总推力就是他们推力的向量和。
- **预算规划**：将收入和支出视为数字向量。组合它们只是向量加法。

#### 为什么这很重要

仅凭这两个运算——加法和缩放——你就可以描述直线、平面乃至整个空间。任何通过组合影响而增长的系统，如物理学、经济学或机器学习，都建立在这些简单规则之上。稍后，当我们定义矩阵乘法、点积和特征值时，它们都可以归结为重复的向量加法和缩放模式。

2. 将 $(1, –2)$ 乘以 3，然后加上 $(0, 5)$。最终向量是什么？

这种简单的练习向你展示了如何通过加法和缩放组合两个基本向量来生成整个结构化空间，这是线性代数真正力量的第一瞥。

#### 参考答案与详细解析

**1. 题目：相加 $(2, 3)$ 和 $(–1, 4)$。**

**解答过程：**
根据向量加法的坐标定义，我们将对应分量相加：
$$
(2, 3) + (-1, 4) = (2 + (-1), \; 3 + 4) = (1, 7).
$$
**几何画法提示**：
1. 从原点 $(0,0)$ 画箭头指向 $(2,3)$。
2. 从 $(2,3)$ 点开始，画一个代表 $(-1,4)$ 的箭头（即向左 1 格，向上 4 格）。
3. 该箭头的终点即为 $(1,7)$。
4. 或者，从原点直接画箭头指向 $(1,7)$，这就是合向量。

**Python 验证：**
```python
import numpy as np
v1 = np.array([2, 3])
v2 = np.array([-1, 4])
print(v1 + v2)  # 输出：[1 7]
```

---

**2. 题目：将 $(1, –2)$ 乘以 3，然后加上 $(0, 5)$。**

**解答过程：**
这是一个线性组合的问题，分两步进行：
第一步，标量乘法：
$$
3 \cdot (1, -2) = (3 \cdot 1, \; 3 \cdot (-2)) = (3, -6).
$$
第二步，向量加法：
$$
(3, -6) + (0, 5) = (3 + 0, \; -6 + 5) = (3, -1).
$$
最终向量是 $(3, -1)$。

**Python 验证：**
```python
v = np.array([1, -2])
scaled = 3 * v
offset = np.array([0, 5])
result = scaled + offset
print(result)  # 输出：[ 3 -1]
```

---

**3. 题目：设 $\mathbf{u} = (1, 2)$ 和 $\mathbf{v} = (2, –1)$。 sketch 所有形式为 $a \cdot \mathbf{u} + b \cdot \mathbf{v}$ 的向量，其中 $a, b \in \{-2, -1, 0, 1, 2\}$。**

**解答过程：**
我们需要计算不同整数对 $(a, b)$ 下的线性组合。
通式为：
$$
a \cdot (1, 2) + b \cdot (2, -1) = (a + 2b, \; 2a - b).
$$

让我们计算几个关键点：
- 当 $a=0, b=0$ 时：$(0, 0)$ （原点）
- 当 $a=1, b=0$ 时：$(1, 2)$ （即向量 $\mathbf{u}$）
- 当 $a=0, b=1$ 时：$(2, -1)$ （即向量 $\mathbf{v}$）
- 当 $a=1, b=1$ 时：$(1+2, 2-1) = (3, 1)$
- 当 $a=-1, b=1$ 时：$(-1+2, -2-1) = (1, -3)$
- ...以此类推。

**观察结果**：
如果你将所有计算出的点画在坐标纸上，你会发现它们形成了一个倾斜的网格。这是因为 $\mathbf{u}$ 和 $\mathbf{v}$ 不共线（线性无关），它们的线性组合可以覆盖整个二维平面。这里的整数网格展示了由这两个基向量生成的离散点阵。

**Python 生成所有点：**
```python
import numpy as np
import matplotlib.pyplot as plt

u = np.array([1, 2])
v = np.array([2, -1])

points = []
# 遍历 a 和 b 从 -2 到 2
for a in range(-2, 3):
    for b in range(-2, 3):
        point = a * u + b * v
        points.append(point)

points = np.array(points)

# 打印前 5 个点示例
print("部分生成点坐标：")
print(points[:5])

# 如果有绘图环境，可以使用以下代码可视化
# plt.scatter(points[:, 0], points[:, 1])
# plt.axhline(0, color='black', linewidth=1)
# plt.axvline(0, color='black', linewidth=1)
# plt.grid(True)
# plt.show()
```
**注意**：这个练习直观地展示了“张成”（Span）的概念。如果 $\mathbf{u}$ 和 $\mathbf{v}$ 方向不同，它们的线性组合可以到达平面上的任何位置。
