### 33. 张成与生成集

张成（Span）的概念捕捉了从旧向量构建新向量最简单且最强大的方法：通过取线性组合。张成不仅仅是一组散乱的点，而是给定向量集所有组合构成的结构化、完整的集合。理解张成可以直接引出基（Basis）、维数（Dimension）以及子空间结构的概念。

#### 张成的定义

给定向量 $v_1, v_2, \dots, v_k \in V$，这些向量的张成定义为：

$$
\text{span}\{v_1, v_2, \dots, v_k\} = \{a_1 v_1 + a_2 v_2 + \dots + a_k v_k : a_i \in \mathbb{R}\}.
$$

- 张成是这些向量所有可能的线性组合构成的集合。
- 它始终是一个子空间（Subspace）。
- 给定的向量集被称为生成集（Generating Set）。

> **注意**：张成集必然包含零向量（当所有系数 $a_i = 0$ 时）。无论给定的向量是什么，它们的张成空间总是通过原点的。

#### 简单示例

1. 在 $\mathbb{R}^2$ 中：

   - $(1,0)$ 的张成 = x 轴的所有倍数（一条直线）。
   - $(1,0)$ 和 $(0,1)$ 的张成 = 整个平面 $\mathbb{R}^2$。
   - $(1,0)$ 和 $(2,0)$ 的张成 = 仍然是 x 轴，因为第二个向量是冗余的。

2. 在 $\mathbb{R}^3$ 中：

   - 单个向量的张成 = 一条直线。
   - 两个线性无关向量的张成 = 通过原点的平面。
   - 三个线性无关向量的张成 = 整个空间 $\mathbb{R}^3$。

我们可以使用 Python 的 `numpy` 库来数值验证向量的张成维度（即矩阵的秩）：

```python
import numpy as np

# 定义向量
v1 = np.array([1, 0])
v2 = np.array([0, 1])
v3 = np.array([2, 0])

# 构建矩阵，每一列是一个向量
A1 = np.column_stack((v1, v2))
A2 = np.column_stack((v1, v3))

# 计算矩阵的秩，秩等于张成空间的维数
rank1 = np.linalg.matrix_rank(A1)
rank2 = np.linalg.matrix_rank(A2)

print(f"Span{{(1,0), (0,1)}} 的维数：{rank1}")  # 输出 2 (整个平面)
print(f"Span{{(1,0), (2,0)}} 的维数：{rank2}")  # 输出 1 (一条直线)
```

#### 张成作为覆盖范围

- 如果将向量视为“方向”，那么张成就是你在这些方向上行走所能到达的所有地方，步长（标量）可以是任意值。
- 如果只有一个方向，你只能在这条线上来回行走。
- 如果有两个独立的方向，你可以扫出一个平面。
- 如果在 3D 中有三个独立的方向，你可以移动到任何地方。

#### 生成集

如果一组向量的张成等于某个子空间，则称这组向量是该子空间的生成集（或张成集）。

- 示例：$\{(1,0), (0,1)\}$ 生成 $\mathbb{R}^2$。
- 示例：$\{(1,0,0), (0,1,0), (0,0,1)\}$ 生成 $\mathbb{R}^3$。
- 示例：矩阵的列向量生成其列空间（Column Space）。

不同的生成集可以张成相同的空间。有些可能是冗余的，有些则是最小的。随后，基（Basis）的概念将 refined 这一想法。

#### 生成集中的冗余性

- 如果一个向量是其他向量的线性组合，它不会扩大张成空间。
- 示例：在 $\mathbb{R}^2$ 中，$\{(1,0), (0,1), (1,1)\}$ 与 $\{(1,0), (0,1)\}$ 张成相同的空间。
- 消除冗余性可以得到更高效的生成集。

> **提示**：判断冗余性的关键在于检查向量组是否线性相关。如果线性相关，则存在冗余；如果线性无关，则每个向量都对张成空间的维数有贡献。

#### 张成与线性方程组

考虑方程组 $Ax=b$。

- 问题“是否有解？”等价于"$b$ 是否在 $A$ 的列向量的张成中？”
- 因此，张成为可解性提供了几何语言。

#### 日常类比

- **语言**：如果你有足够的字母（生成元），你可以拼出任何单词（向量）。冗余的字母不会增加能力（多余的"A"没有帮助）。
- **食谱**：食材就像生成元；张成是你用它们能做的所有菜肴。有些食材是必不可少的，有些可以去掉。
- **旅行方向**：如果你只知道如何向北走，你被困在一条线上。加上向东走，你就可以到达地图上的任何地方。

#### 为什么它很重要

1. 张成是定义由向量生成的子空间的基础。
2. 它直接联系到线性方程组的可解性。
3. 它引入了冗余的概念，为基和线性无关性做准备。
4. 它自然地推广到函数空间和抽象向量空间。

#### 练习题解答与解析

##### 1. 求 $\{(1,2), (2,4)\}$ 在 $\mathbb{R}^2$ 中的张成

**解答过程：**
观察这两个向量 $v_1 = (1,2)$ 和 $v_2 = (2,4)$。
我们可以发现 $v_2 = 2 \cdot v_1$。
这意味着这两个向量是线性相关的（共线）。
因此，它们的线性组合 $a_1 v_1 + a_2 v_2$ 实际上可以写成 $(a_1 + 2a_2)v_1$。
令 $k = a_1 + 2a_2$，则任意组合都是 $k(1,2)$ 的形式。

**结论：**
张成是通过原点和点 $(1,2)$ 的一条直线。即集合 $\{ (x, y) \in \mathbb{R}^2 \mid y = 2x \}$。

**Python 验证：**
```python
import numpy as np

A = np.array([[1, 2], 
              [2, 4]])
# 计算秩
rank = np.linalg.matrix_rank(A)
print(f"矩阵的秩：{rank}")  # 输出 1，说明张成空间维数为 1（直线）
```

##### 2. 证明向量 $(1,0,1), (0,1,1), (1,1,2)$ 在 $\mathbb{R}^3$ 中仅张成一个平面

**解答过程：**
设 $v_1 = (1,0,1)$, $v_2 = (0,1,1)$, $v_3 = (1,1,2)$。
我们要检查这三个向量是否线性无关。观察发现：
$$
v_1 + v_2 = (1,0,1) + (0,1,1) = (1, 1, 2) = v_3
$$
由于 $v_3$ 是 $v_1$ 和 $v_2$ 的线性组合，它是冗余的。
因此，$\text{span}\{v_1, v_2, v_3\} = \text{span}\{v_1, v_2\}$。
因为 $v_1$ 和 $v_2$ 不成比例（线性无关），它们张成一个二维子空间。
在 $\mathbb{R}^3$ 中，通过原点的二维子空间就是一个平面。

**结论：**
这三个向量张成的是一个平面，而不是整个 $\mathbb{R}^3$ 空间。

**Python 验证：**
```python
import numpy as np

vecs = np.array([[1, 0, 1], 
                 [0, 1, 1], 
                 [1, 1, 2]]).T  # 转置使得每列为一个向量

rank = np.linalg.matrix_rank(vecs)
print(f"向量组的秩：{rank}")  # 输出 2，说明张成空间维数为 2（平面）
```

##### 3. 判断 $(1,2,3)$ 是否在 $(1,0,1)$ 和 $(0,1,2)$ 的张成中

**解答过程：**
设 $v_1 = (1,0,1)$, $v_2 = (0,1,2)$, $b = (1,2,3)$。
我们需要判断是否存在标量 $c_1, c_2$ 使得：
$$
c_1 \begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix} + c_2 \begin{pmatrix} 0 \\ 1 \\ 2 \end{pmatrix} = \begin{pmatrix} 1 \\ 2 \\ 3 \end{pmatrix}
$$
这等价于求解线性方程组：
1. $c_1 = 1$
2. $c_2 = 2$
3. $c_1 + 2c_2 = 3$

将前两个方程的解代入第三个方程检验：
$1 + 2(2) = 1 + 4 = 5 \neq 3$。
方程组不一致，无解。

**结论：**
向量 $(1,2,3)$ **不在** 给定向量的张成中。

**Python 验证：**
```python
import numpy as np

A = np.array([[1, 0], 
              [0, 1], 
              [1, 2]])
b = np.array([1, 2, 3])

# 使用最小二乘法求解，如果残差很大则无精确解
x, residuals, rank, s = np.linalg.lstsq(A, b, rcond=None)
reconstructed_b = A @ x

print(f"重构的 b: {reconstructed_b}")
print(f"原始 b:   {b}")
print(f"是否相等：{np.allclose(reconstructed_b, b)}")  # 输出 False
```

##### 4. 挑战：证明所有多项式的集合 $\{1, x, x^2, \dots\}$ 张成所有多项式的空间

**解答过程：**
这是一个关于无限维向量空间的问题。
1. **定义多项式空间**：所有多项式的空间 $P$ 包含形如 $p(x) = a_0 + a_1 x + a_2 x^2 + \dots + a_n x^n$ 的对象，其中 $n$ 是任意非负整数，$a_i$ 是系数。
2. **生成集的定义**：集合 $S = \{1, x, x^2, \dots\}$。
3. **张成的含义**：$\text{span}(S)$ 包含 $S$ 中元素的所有**有限**线性组合。
4. **论证**：
   对于任意多项式 $p(x) \in P$，根据定义，它只有有限项非零系数。
   因此，任何 $p(x)$ 都可以写成：
   $$
   p(x) = a_0 \cdot 1 + a_1 \cdot x + a_2 \cdot x^2 + \dots + a_n \cdot x^n
   $$
   这正是集合 $S$ 中前 $n+1$ 个元素的线性组合。
   因为任意多项式都可以表示为 $S$ 中元素的线性组合，所以 $S$ 张成整个多项式空间。

**结论：**
集合 $\{1, x, x^2, \dots\}$ 是多项式空间的标准生成集（实际上也是标准基）。

张成的概念改变了我们的视角：我们不再关注单个向量，而是看到它们生成的整个可能性 landscape。
